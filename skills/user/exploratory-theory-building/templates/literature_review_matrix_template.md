## 記入ガイドライン

### 1. 著者（年）
- **形式**：姓, イニシャル（年）
- **例**：Eisenhardt, K. M. (1989)
- **複数著者**：第1著者 et al. (年)

### 2. タイトル
- **形式**：原題（英語論文の場合）
- **日本語論文**：日本語タイトル
- **注意**：正確に転記（後でZoteroと照合）

### 3. 掲載誌
- **形式**：雑誌名（正式名称）
- **例**：Academy of Management Journal
- **注意**：略称ではなく正式名称を使用

### 4. 理論レンズ
研究が採用している主要な理論的視点を記録：
- **RBV**（Resource-Based View）
- **TCE**（Transaction Cost Economics）
- **Institutional Theory**
- **Dynamic Capabilities**
- **Knowledge-Based View**
- **Agency Theory**
- **その他**：[具体的に記述]

### 5. 研究質問
- **形式**：著者が明示的に述べている研究質問
- **ない場合**：論文から推測される中核的問い
- **注意**：1-2文で簡潔に

### 6. 方法論
研究デザインとデータ収集・分析方法：
- **定性**：ケーススタディ、インタビュー、エスノグラフィー
- **定量**：サーベイ、アーカイブデータ、実験
- **混合**：定性と定量の統合
- **概念的**：理論的議論のみ（実証研究なし）

### 7. 主要発見
- **形式**：2-3文で最も重要な発見を要約
- **注意**：著者の主張をそのまま記録（自分の解釈は「メモ」欄へ）

### 8. 理論的貢献
論文が既存知識に付加した新規性：
- **新概念の提案**：[概念名]
- **既存理論の拡張**：[どのように拡張したか]
- **矛盾の解決**：[どのような矛盾を解決したか]
- **新たな視点**：[どのような視点を提供したか]

### 9. 限界
- **著者が認める限界**：論文の "Limitations" セクションから抽出
- **批判的評価**：あなた自身が見出した限界
- **注意**：建設的批判の視点を保持

### 10. 引用数
- **出典**：Google Scholar
- **形式**：概数（例：5,000+）
- **注意**：高引用=高品質ではないが、影響力の指標

### 11. 関連性
自己の研究との関連性を3段階で評価：
- **高**：直接的に関連、必ず引用すべき
- **中**：間接的に関連、文脈によって引用
- **低**：周辺的関連、参考程度

### 12. メモ
- **自由記述欄**
- **記録すべき内容**：
  - 印象的な引用
  - 自己研究への示唆
  - 批判的コメント
  - 今後読むべき関連文献
  - 理論的アイデア

## 理論的ギャップの特定

文献レビュー表から、以下の4類型の理論的ギャップを特定してください：

### Gap Type 1：未探索現象（Unexplored Phenomenon）
- **定義**：既存理論では説明されていない新しい現象
- **特定方法**：「このテーマを扱った研究はあるか？」
- **例**：「デジタル変革期の競争優位」が2015年以前は未研究

### Gap Type 2：理論的矛盾（Theoretical Contradiction）
- **定義**：既存研究間の矛盾・不一致
- **特定方法**：「主要発見」列を比較し、矛盾を探す
- **例**：研究Aは正の効果、研究Bは無関係を報告

### Gap Type 3：境界条件不明（Boundary Condition Unclear）
- **定義**：理論の適用範囲が不明確
- **特定方法**：「いつ・どこで成立するか？」が明示されていない
- **例**：RBVはいつ成立するか？産業・国・時代による違いは？

### Gap Type 4：メカニズム不明（Mechanism Underspecified）
- **定義**：「AはBに影響する」は示されているが、「なぜ・どのように」が不明
- **特定方法**：因果関係は示されているが、媒介・調整メカニズムが不明
- **例**：リーダーシップ→業績の媒介メカニズム（モチベーション？調整？）

## 参考：優れた文献レビューの特徴

1. **網羅性**：主要研究を漏れなくカバー
2. **批判性**：各研究の限界を指摘
3. **統合性**：研究単位ではなく、テーマで統合
4. **理論的焦点**：単なる要約ではなく、理論的貢献に焦点
5. **ギャップの明確化**：既存知識の限界を明示
6. **自己研究の位置づけ**：自分の研究がどのギャップを埋めるか明確化
