# 引用スタイルの詳細例文

## 本文内引用の実践例

### パターン1：基本的な引用

```
　Porter（1980）の競争戦略論は、企業が持続的競争優位を獲得する
ためには、コストリーダーシップか差別化のいずれかの戦略を選択する
必要があると主張した（Porter, 1980, p.35）。
```

### パターン2：複数の研究を同時に引用

```
　プラットフォームビジネスの成功要因については、多くの研究が
蓄積されてきた（Parker et al., 2016; Eisenmann et al., 2006; 
Gawer & Cusumano, 2014）。
```

### パターン3：データの引用

```
　日本の製造業全体の平均営業利益率は4.2パーセントに留まっており
（経済産業省, 2023, p.25）、グローバル競争の激化により収益性の
向上は喫緊の課題となっている。
```

### パターン4：企業データの引用

```
　キーエンスの2023年度の営業利益率は51.1パーセントに達している
（キーエンス, 2024, p.3）。同社の時価総額は約15兆円に達し、
日本企業の中でトヨタ自動車に次ぐ第2位の規模である（日本経済新聞, 
2023）。
```

### パターン5：複数ページにわたる引用

```
　Booking Holdingsは2005年のBooking.com買収を皮切りに、積極的な
M&A戦略を展開してきた（Booking Holdings, 2024, pp.12-15）。
```

### パターン6：再引用（孫引き）

```
　Barneyの初期の研究では、資源の価値は文脈依存的であると指摘
されていた（Barney, 1986, Porter, 1991より引用, p.105）。
```

### パターン7：同一著者の複数文献

```
　田中は一連の研究において、日本企業の国際化戦略を分析してきた
（田中, 2020a; 田中, 2020b; 田中, 2021）。
```

### パターン8：本文中に著者名が出現

```
　Porterは、競争戦略の本質は企業がいかにして持続的な競争優位を
構築するかにあると述べている（Porter, 1980, p.35）。

　あるいは

　Porter（1980, p.35）は、競争戦略の本質は企業がいかにして
持続的な競争優位を構築するかにあると述べている。
```

## 引用の実践的テクニック

### テクニック1：引用の密度管理

**適切な例（先行研究レビュー）**：
```
　プラットフォームビジネスの競争戦略については、多くの研究が
蓄積されている。Parker et al.（2016）は、プラットフォームが
多面市場を形成し、間接ネットワーク効果により指数関数的成長を
実現することを明らかにした。Eisenmann et al.（2006）は、
プラットフォーム戦略における価格設定の重要性を論じ、補完財
提供者への補助により需要側ネットワーク効果を増幅できることを
示した。また、Gawer & Cusumano（2014）は、プラットフォーム
リーダーシップの概念を提示し、エコシステム管理の重要性を
強調した。
```

**引用が不足している例**：
```
　プラットフォームビジネスは、多面市場を形成し、ネットワーク
効果により成長する。価格設定が重要であり、エコシステム管理も
必要である。
```

### テクニック2：データには必ず引用

**適切な例**：
```
　日本の製造業全体の平均営業利益率は4.2パーセントであり
（経済産業省, 2023, p.25）、これは米国の5.8パーセント、ドイツの
6.2パーセントと比較して低い水準にある（OECD, 2023, p.42）。
```

**引用が不足している例**：
```
　日本の製造業の営業利益率は約4パーセント程度で、欧米と比較して
低い水準にある。
```

### テクニック3：理論の正確な引用

**適切な例**：
```
　Barney（1991）の資源ベース理論によれば、持続的競争優位の
源泉となる資源は、VRIN条件、すなわち、貴重性（Valuable）、
希少性（Rare）、模倣困難性（Inimitable）、非代替性
（Non-substitutable）の四つの条件を満たす必要がある（Barney, 
1991, pp.105-106）。
```

**不正確な例**：
```
　資源ベース理論によれば、競争優位には独自の資源が必要である。
```

### テクニック4：引用の統合

**適切な例（複数の研究を統合）**：
```
　デジタル変革の成功要因については、リーダーシップ（Westerman 
et al., 2014）、組織文化（Kane et al., 2015）、ITガバナンス
（Weill & Ross, 2004）、デジタル人材（Singh & Hess, 2017）
など、複数の視点から研究されてきた。
```

### テクニック5：批判的引用

**適切な例**：
```
　Porter（1980）の競争戦略論は、コストリーダーシップと差別化は
相互に排他的な戦略であると主張した。しかし、近年の研究は、
両者の統合が可能であることを示唆している（Hill, 1988; 
Murray, 1988）。
```

## よくある引用の誤り

### 誤り1：引用のない主張

**❌ 悪い例**：
```
　多くの研究者が指摘しているように、プラットフォームビジネスは
重要である。
```

**✅ 良い例**：
```
　プラットフォームビジネスの戦略的重要性については、多くの研究者
が指摘している（Parker et al., 2016; Eisenmann et al., 2006）。
```

### 誤り2：不正確な引用形式

**❌ 悪い例**：
```
（Porter 1980）
（Porter M.E., 1980）
（Porter, M.E. (1980)）
```

**✅ 良い例**：
```
（Porter, 1980）
（Porter, 1980, p.35）
```

### 誤り3：ページ番号の欠落

**❌ 悪い例**：
```
　Porterは「競争の本質は収益性である」と述べている（Porter, 
1980）。
```

**✅ 良い例**：
```
　Porterは「競争の本質は収益性である」と述べている（Porter, 
1980, p.4）。
```

### 誤り4：二次引用の不適切な使用

**❌ 悪い例**：
```
　Barney（1986）は資源の価値は文脈依存的であると指摘した
（Porter, 1991より）。
```

**✅ 良い例**：
```
　Barneyの初期の研究では、資源の価値は文脈依存的であると指摘
されていた（Barney, 1986, Porter, 1991より引用, p.105）。

【注】可能な限り原典を確認し、直接引用することが望ましい。
```

### 誤り5：参考文献リストとの不一致

本文で引用した文献は、必ず参考文献リストに記載してください。
逆に、参考文献リストに記載した文献は、本文中で最低1回は引用して
ください。
