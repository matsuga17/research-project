# Academic Paper Creation Skill

## 概要

このスキルは、30,000字規模の本格的な日本語学術論文を作成するための包括的ガイドラインです。新規理論の構築、実証研究設計、厳格な東京大学引用スタイルの管理を重視し、国際的な学術水準に準拠した論文を生成します。

## 主な特徴

### 📝 論述形式の徹底
- 論文本文での箇条書きを完全排除
- すべて論述形式（段落形式）で記述
- 学術論文として適切な文章構造

### 🎯 新規理論構築支援
- 既存研究の空白を特定
- 検証可能な仮説の導出
- 理論的貢献の明確化

### 📚 東京大学引用スタイル完全準拠
- 本文内引用：（著者, YYYY, p.XX）形式
- 20種類以上の引用パターンに対応
- 参考文献リストの自動整形

### 🔬 実在データ原則
- 架空データの使用を厳禁
- 検証可能な公開データのみ使用
- データの出典を厳密に管理

## 適用対象

### ✅ 推奨される用途
- 経営学・経済学・社会科学分野の学術論文
- 30,000字規模（28,000-32,000字）の本格的研究論文
- 新規理論構築を伴う研究
- 実証研究・事例分析
- 査読論文の執筆

### ❌ 適用外
- 短文レポート（5,000字未満）
- ビジネス文書、プレゼンテーション資料
- 創作的文章（エッセイ、小説）
- 技術マニュアル

## 論文構成（30,000字）

| セクション | 文字数 | 内容 |
|-----------|--------|------|
| 序論 | 2,500-3,000 | 問題意識、研究目的、意義 |
| 先行研究 | 3,500-4,000 | 既存研究の整理、理論的空白 |
| 理論枠組み | 3,000-3,500 | 分析の理論的基盤 |
| 研究方法 | 2,500-3,000 | 研究設計、データ収集 |
| 実証分析 | 9,000-10,000 | データ分析、事例検討 |
| 理論構築 | 4,000-4,500 | 仮説検証、新規理論提示 |
| 考察 | 3,000-3,500 | 学術的・実務的意義 |
| 結論 | 1,500-2,000 | 要約、今後の課題 |
| 参考文献 | 1,000-1,500 | 30-50文献 |

## 使用方法

### 基本的な使い方

1. **研究テーマの指定**
```
「キーエンスの超高収益モデルに関する30,000字の学術論文を作成してください」
```

2. **自動実行**
- Phase 1: 重要度判定・構想（7-10分）
- Phase 2: 情報収集・分析（15-25分）
- Phase 3: 論文執筆（8-12分）

3. **品質確認**
- 論述形式の確認
- 引用の正確性確認
- 文字数の確認

### 推奨MCPツール

- **web_search**: 最新情報の収集
- **omnisearch**: 学術的・信頼性の高い情報
- **sequential-thinking**: 複雑な問題の構造化
- **think-tank**: 知識統合・洞察生成

## 品質基準

### 必須要件（形式）
- ✅ 箇条書きが本文で使用されていない
- ✅ 全段落で字下げ（全角スペース1文字）
- ✅ である調で統一
- ✅ 東京大学引用スタイル準拠
- ✅ すべての引用に出典明記

### 必須要件（内容）
- ✅ 新規性の高い理論的貢献
- ✅ 実在データのみ使用
- ✅ 先行研究の空白を特定
- ✅ 研究の限界を誠実に記述
- ✅ 論理的整合性

## 例文サンプル

### 論述形式の正しい例

**悪い例（箇条書き）**：
```
成功要因は以下の三つである。
• ファブレス製造
• 直接販売
• コンサルティング営業
```

**良い例（論述形式）**：
```
　成功要因は三つの次元から説明できる。第一に、ファブレス製造により
製造固定費を最小化することで、価格設定の柔軟性が確保される。第二に、
直接販売体制により顧客ニーズが即座に製品開発にフィードバックされる
情報循環が実現している。第三に、コンサルティング営業により顧客の
潜在的課題を解決する高付加価値提案が可能となっている。
```
