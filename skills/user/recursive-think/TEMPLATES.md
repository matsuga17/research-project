## 品質チェックリスト

### 完了前の最終確認

```markdown
## RecursiveThink 品質チェックリスト

### エージェント参加度
- [ ] 体系的分析者：発言[N]回
- [ ] 創造的探索者：発言[N]回
- [ ] 批判的評価者：発言[N]回
- [ ] 統合的総合者：発言[N]回
- [ ] 実践的戦略家：発言[N]回
- [ ] 引用検証専門家：発言[N]回
- [ ] すべてのエージェントが少なくとも1回発言

### 段階完了度
- [ ] 第0段階：完了（構造化マップ、概念定義、初期引用）
- [ ] 第1段階：完了（批判的検討、論点抽出、代替仮説）
- [ ] 第2段階：完了（エビデンス収集、引用検証、信頼性評価）
- [ ] 第3段階：完了（統合フレームワーク、実装計画、最終レポート）

### 思考劣化防止
- [ ] 反論生成プロトコル：実施[N]回
- [ ] 視点シフト：実施[N]回
- [ ] 思考リセット：実施[N]回（必要に応じて）
- [ ] 同じ結論の繰り返し：なし

### 引用・参考文献
- [ ] すべての引用が東京大学スタイルに準拠
- [ ] URLがすべて検証済み
- [ ] リンク切れに代替URLを提供
- [ ] 情報源の信頼性評価が完了
- [ ] 25語以下の引用制限遵守
- [ ] 1ソース1引用原則遵守

### 出力品質
- [ ] アーティファクト形式で生成
- [ ] エグゼクティブサマリー含む
- [ ] 段階別分析結果が明確
- [ ] エージェント間対話が記録
- [ ] メタ認知的評価が含まれる
- [ ] 次のステップが明示

### 実用性
- [ ] 実装可能な具体的提案
- [ ] リスクと緩和策が明示
- [ ] リソース要件が明確
- [ ] スケジュールが現実的

### 総合評価
- 分析の深さ：[1-5]
- 多様性：[1-5]
- 厳密性：[1-5]
- 実用性：[1-5]
- 革新性：[1-5]

**総合スコア**：[合計/25]
- 20-25：卓越
- 15-19：推奨基準達成
- 10-14：最低基準達成
- 9以下：要改善
```
