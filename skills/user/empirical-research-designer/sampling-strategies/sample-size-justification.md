# サンプルサイズ根拠：Sample Size Justification

## 概要

適切なサンプルサイズの決定と根拠の提示は、量的研究の**方法論的厳密性**の基盤である。サンプルサイズが不十分であれば統計的検出力が低下し、過剰であれば資源の浪費となる。本ガイドは、統計的検出力分析、実践的ルール、研究デザイン別推奨値を統合的に提示する。

---

## I. 理論的基盤

### 1.1 統計的検出力分析（Power Analysis）

**検出力（Power）の定義**

統計的検出力（1-β）とは、真の効果が存在する時にそれを正しく検出する確率である。

**4つの相互関連要素**

1. **サンプルサイズ（n）**：研究者が制御可能
2. **効果量（Effect Size, δ）**：現象の実質的大きさ
3. **有意水準（α）**：第一種過誤の確率（通常0.05）
4. **検出力（1-β）**：第二種過誤の確率（通常0.80）

**これら4要素は数学的に結びついており、3つが決まれば残り1つが決定される。**

### 1.2 効果量（Effect Size）

効果量は、現象の実質的大きさを標準化した指標であり、統計的有意性とは独立に解釈可能である。

**Cohenの効果量基準**（Cohen 1988）

#### 平均値差の効果量（Cohen's d）

```
d = (μ₁ - μ₂) / σ

d = 0.2  小効果（small）
d = 0.5  中効果（medium）
d = 0.8  大効果（large）
```

**例**：教育介入の効果
- 介入群平均：75点、統制群平均：70点、標準偏差：10点
- d = (75-70)/10 = 0.5 → 中効果

#### 相関係数の効果量（r）

```
r = 0.1  小効果
r = 0.3  中効果
r = 0.5  大効果
```

#### 回帰分析の効果量（f²）

```
f² = R²/(1-R²)

f² = 0.02  小効果
f² = 0.15  中効果
f² = 0.35  大効果
```

**効果量の推定方法**

1. **先行研究のメタ分析**：最も信頼性高い
2. **パイロットスタディ**：予備的データから推定
3. **理論的予想**：実質的に意味のある最小効果（SESOI: Smallest Effect Size of Interest）
4. **Cohenの基準値**：上記が不可能な場合の保守的推定（中効果 d=0.5）

**戦略経営・組織論における典型的効果量**（実証的知見）

- 組織文化 → 業績：r = 0.15-0.30（小〜中）
- リーダーシップ → 従業員満足度：d = 0.40-0.70（中〜大）
- AI投資 → 生産性向上：f² = 0.10-0.25（小〜中）

**重要**：社会科学では大効果（d=0.8, r=0.5）は稀。多くの実証研究で小〜中効果を扱う。

## II. 研究デザイン別サンプルサイズ計算

### 2.1 t検定（2群比較）

**独立サンプルt検定**

```
必要サンプルサイズ（各群）：

n = 2(Z_α/2 + Z_β)² × σ² / δ²

標準設定：
α = 0.05 → Z_α/2 = 1.96
β = 0.20 → Z_β = 0.84 (検出力0.80)
```

**簡易計算表**（両側検定、α=0.05、検出力=0.80）

| 効果量 | 各群サンプル数 | 合計サンプル数 |
|-------|--------------|--------------|
| d = 0.2（小） | 393 | 786 |
| d = 0.5（中） | 64 | 128 |
| d = 0.8（大） | 26 | 52 |

**例**：AI導入トレーニングの効果検証
- 予想効果量：d = 0.5（中効果）
- 必要サンプル：各群64名、合計128名
- 回収率60%想定 → 配布数：各群107名、合計214名

**対応サンプルt検定**（事前-事後比較）

対応サンプルでは、個人内相関により必要サンプル数が減少。

```
n_対応 = n_独立 × (1-ρ) / 2

ρ：事前-事後の相関係数
```

**例**
- 独立サンプルで必要n = 64
- 事前-事後相関 ρ = 0.50 の場合
- n_対応 = 64 × (1-0.50) / 2 = 16

### 2.2 分散分析（ANOVA）

**一元配置ANOVA**（3群以上の比較）

```
n = λ × df_error / (k-1) + k

λ：非心度パラメータ（効果量・α・検出力から決定）
k：群数
df_error：誤差自由度
```

**簡易計算表**（α=0.05、検出力=0.80）

| 効果量 f | 群数3 | 群数4 | 群数5 |
|---------|------|------|------|
| 0.10（小） | 322 | 274 | 245 |
| 0.25（中） | 52 | 45 | 40 |
| 0.40（大） | 21 | 18 | 16 |

**二元配置ANOVA**

主効果・交互作用それぞれについて検出力分析が必要。交互作用の検出には、主効果の約4倍のサンプルが必要（McClelland & Judd 1993）。

### 2.3 相関分析

**ピアソン相関係数の有意性検定**

```
n = (Z_α/2 + Z_β)² / C² + 3

C = 0.5 × ln((1+r)/(1-r)) （Fisherのz変換）
```

**簡易計算表**（α=0.05、検出力=0.80）

| 相関係数 r | 必要サンプル数 |
|-----------|--------------|
| 0.10（小） | 782 |
| 0.20 | 193 |
| 0.30（中） | 84 |
| 0.40 | 46 |
| 0.50（大） | 28 |

**例**：組織文化と業績の相関
- 予想相関：r = 0.30（中効果）
- 必要サンプル：84社

### 2.4 重回帰分析

**必要サンプルサイズの実践的ルール**

**基本ルール**（Green 1991）

```
最小サンプルサイズ：

n ≥ 50 + 8×k  （全体モデル検定）
n ≥ 104 + k   （個別係数検定）

k：独立変数の数
```

**推奨ルール**（Tabachnick & Fidell 2013）

```
n ≥ 15×k  （推奨最小）
n ≥ 20×k  （理想的）
```

**例**：企業業績の規定要因分析
- 独立変数：10個（AI投資、R&D、規模、年齢等）
- 最小サンプル：50 + 8×10 = 130社
- 推奨サンプル：15×10 = 150社
- 理想的サンプル：20×10 = 200社

**検出力分析によるサンプルサイズ**（Cohen 1988）

| 効果量 f² | 検出力0.80 | 検出力0.90 |
|----------|-----------|-----------|
| 0.02（小） | 約500 + 10k | 約650 + 10k |
| 0.15（中） | 約70 + 5k | 約90 + 5k |
| 0.35（大） | 約35 + 3k | 約45 + 3k |

### 2.5 構造方程式モデリング（SEM）

**一般的推奨値**

| 基準 | 推奨サンプル数 | 根拠 |
|------|--------------|------|
| 絶対最小 | 100-150 | Ding et al. (1995) |
| 推奨最小 | 200 | Kline (2016) |
| 理想的 | 400以上 | Boomsma (1982) |

**パラメータ数基準**

```
n ≥ 5×p  （最小）
n ≥ 10×p （推奨）
n ≥ 20×p （理想）

p：推定パラメータ数
```

**例**：5構成概念、各3-5項目のモデル
- 推定パラメータ：約30-50
- 最小サンプル：5×40 = 200
- 推奨サンプル：10×40 = 400

**複雑性による調整**（Kline 2016）

- **単純なモデル**（3-4構成概念、良好な測定）：200-300
- **中程度の複雑性**（5-7構成概念、標準的測定）：300-500
- **複雑なモデル**（8+構成概念、複雑な構造）：500-1000+

**モデル適合度とサンプルサイズ**

サンプルサイズが大きいほど（n>500）、χ²検定が有意になりやすい。RMSEA、CFI、TLIなどの近似適合度指標を重視。

### 2.6 マルチレベル分析

マルチレベル分析（HLM）では、レベル1（個人）とレベル2（組織）の両方のサンプルサイズが重要。

**推奨基準**（Maas & Hox 2005）

- **レベル2（組織数）**：最低30、推奨50以上
- **レベル1（個人数/組織）**：最低20、推奨30以上

**検出力への影響**

レベル2のサンプル数がより重要。レベル2が30未満の場合、レベル2の効果検出力が著しく低下。

**例**：従業員満足度の組織間差異分析
- レベル2（組織）：50社（推奨）
- レベル1（従業員/組織）：30名
- 合計サンプル：50×30 = 1,500名

## III. 実践的考慮事項

### 3.1 回収率・脱落率の見込み

**現実的回収率**（研究タイプ別）

| 調査方法 | 典型的回収率 | 補正係数 |
|---------|------------|---------|
| 郵送サーベイ | 20-40% | 2.5-5倍 |
| オンラインサーベイ | 30-50% | 2-3.3倍 |
| 対面インタビュー | 70-90% | 1.1-1.4倍 |
| 企業調査（B2B） | 15-30% | 3.3-6.7倍 |
| 学生サンプル | 60-80% | 1.25-1.7倍 |

**配布数の計算**

```
配布数 = 必要サンプル数 / 予想回収率

例：必要サンプル200、回収率30%想定
配布数 = 200 / 0.30 = 667
```

**縦断的研究の脱落率**

| 期間 | 典型的脱落率 |
|------|------------|
| 6ヶ月 | 10-20% |
| 1年 | 20-30% |
| 2年 | 30-50% |

**対策**：初期サンプルを脱落率分増やす、インセンティブ、定期的連絡。

### 3.2 予算・時間制約下での決定

**現実的制約**がある場合の優先順位：

**戦略1: 検出力を犠牲にする**（非推奨だが現実的に多い）
- 検出力0.80 → 0.70に引き下げ
- サンプルサイズ約20%削減可能

**戦略2: 効果量を調整する**
- 小効果を諦め、中効果以上のみ検出
- サンプルサイズ大幅削減可能

**戦略3: 研究デザインの変更**
- 対応サンプルデザインへの変更（サンプルサイズ削減）
- 共変量の追加（統計的統制による検出力向上）

**戦略4: 透明な報告**
- 実際のサンプルサイズによる事後検出力（post-hoc power）報告
- 検出可能な最小効果量（minimum detectable effect size）明示
- 結果の解釈における慎重さ

**例**：理想的サンプル500だが、予算的に200が限界
- 事後検出力分析：検出可能な最小効果量 d = 0.40（中効果）
- 報告：「本研究のサンプルサイズ（n=200）では、中効果（d≥0.40）は80%の検出力で検出可能だが、小効果（d=0.20）の検出力は約30%にとどまる。したがって、有意でない結果は、効果が存在しないことではなく、検出力不足による可能性がある」

### 3.3 パイロットスタディの活用

**目的**
- 効果量の予備的推定
- 測定尺度の信頼性・妥当性評価
- 回収率の見積もり

**推奨サンプルサイズ**
- 最小：30-50
- 推奨：50-100
- 本調査の10-20%

**パイロットデータの活用**
1. 効果量の推定 → より正確なサンプルサイズ計算
2. 測定尺度の改善 → 信頼性向上 → 検出力向上
3. 回収率の把握 → 配布数の調整

**注意**：パイロットの効果量推定は変動が大きい。保守的な推定（95%信頼区間の下限値使用）を推奨。

## IV. サンプルサイズ根拠の報告

### 4.1 論文における報告の標準形式

**方法論セクションでの記述例**

> **サンプルサイズ決定**  
> 本研究では、事前検出力分析により必要サンプルサイズを決定した。先行研究のメタ分析（Smith et al., 2020）に基づき、AI投資と企業業績の関係の効果量をf² = 0.15（中効果）と推定した。有意水準α = 0.05、検出力1-β = 0.80、独立変数8個の条件下で、G*Power 3.1による計算の結果、必要サンプルサイズは107社と算出された。回収率を30%と見込み、350社に調査票を配布した。

**より詳細な報告**（必要に応じて）

> **サンプルサイズ決定の根拠**  
> 本研究の主要仮説（H1: AI投資は企業業績に正の影響を与える）の検証に必要なサンプルサイズを、統計的検出力分析により決定した。
> 
> **効果量の推定**：Smith et al. (2020)のメタ分析（k=25研究、N=3,450社）によれば、AI投資と業績の平均効果量はf² = 0.15（95% CI: 0.10-0.22）であった。保守的推定として、信頼区間の下限値f² = 0.10を採用した。
> 
> **検出力設定**：有意水準α = 0.05（両側検定）、検出力1-β = 0.80を設定した。独立変数は8個（AI投資額、R&D投資、企業規模、企業年齢、産業ダミー3つ）である。
> 
> **計算結果**：G*Power 3.1.9.7（Faul et al., 2009）を用いた線形重回帰分析の検出力分析により、必要サンプルサイズは160社と算出された。
> 
> **実際の配布数**：企業調査の典型的回収率20-30%（Cycyota & Harrison, 2006）を考慮し、配布数を533社（160 / 0.30）とした。最終的に178社から回答を得た（回収率33%）。

### 4.2 事後検出力分析（Post-Hoc Power Analysis）の慎重な使用

**事後検出力の問題点**

事後検出力分析（observed power）は、得られたサンプルサイズと観察された効果量に基づく検出力計算だが、**方法論的に問題がある**とされる（Hoenig & Heisey 2001）。

**理由**
- 有意でない結果 → 観察効果量小 → 事後検出力低 → 循環論法
- p値と事後検出力は数学的に1対1対応（追加情報なし）

**推奨される代替**

**1. 感度分析（Sensitivity Analysis）**

実際のサンプルサイズで検出可能な最小効果量（MDES: Minimum Detectable Effect Size）を報告。

**例**
> 本研究のサンプルサイズ（n=120）では、α = 0.05、検出力0.80の条件で、d = 0.52以上の効果を検出可能である。観察された効果量（d = 0.35）は、この閾値を下回っており、統計的有意性に至らなかった可能性がある。

**2. 信頼区間の報告**

効果量の信頼区間を報告し、実質的意味のある効果が除外されていないか検討。

**例**
> AI投資の標準化回帰係数は β = 0.18（95% CI: -0.05, 0.41, p = .12）であった。信頼区間は0を含むが、正の効果（最大0.41）の可能性も完全に排除されない。サンプルサイズの制約により、中程度の効果の存在を結論づけることができなかった。

## V. よくある問題と解決策

### 問題1: サンプルサイズが不十分だと指摘される（査読・審査）

**対応戦略**

**戦略1: 事前検出力分析の根拠を詳述**
- 効果量推定の根拠（先行研究、メタ分析、理論的予想）
- 検出力設定の妥当性
- 制約条件（予算、アクセス困難）の透明な報告

**戦略2: 感度分析の追加**
- 実際のサンプルで検出可能な最小効果量の明示
- 「本研究は小効果の検出には不十分だが、中〜大効果は十分な検出力で検出可能」

**戦略3: 効果量・信頼区間重視の解釈**
- p値だけでなく、効果量と信頼区間を報告
- 実質的意義（practical significance）の議論

**戦略4: 限界の正直な認識**
- 限界セクションで検出力不足を認める
- 将来研究での大規模検証を提案

### 問題2: サンプルサイズが過剰と批判される

**過剰サンプルの問題**
- 統計的有意性が容易に達成され、実質的に意味のない小効果も有意に
- 資源の浪費

**対応**

**戦略1: 効果量重視の解釈**
- p値よりも効果量と信頼区間を強調
- 実質的意義（practical significance）の基準を明示

**例**
> サンプルサイズが大きい（n=1,500）ため、統計的有意性は容易に達成される。したがって、効果の実質的意義を評価するため、効果量（r = 0.08）と信頼区間（95% CI: 0.03, 0.13）に焦点を当てる。Cohenの基準では小効果に相当し、実践的インプリケーションは限定的である。

**戦略2: 大サンプルの利点を強調**
- 小効果の精密な推定
- サブグループ分析の可能性
- 頑健性チェックの実施

### 問題3: 効果量の推定が困難（新規現象、先行研究不足）

**対応戦略**

**戦略1: 保守的推定（中効果）**
- Cohenの中効果（d=0.5, r=0.3, f²=0.15）を仮定
- 保守的だが、正当化可能

**戦略2: 複数シナリオの検討**
- 小・中・大効果の3シナリオで計算
- 中効果を基準として採用し、小・大効果のケースも報告

**例**
> 新規現象であり効果量の推定が困難なため、Cohen (1988)の中効果（f² = 0.15）を仮定した。感度分析として、小効果（f² = 0.02）では必要サンプルn = 500、大効果（f² = 0.35）ではn = 40であることを確認した。

**戦略3: パイロットスタディ実施**
- 小規模予備調査（n=50-100）で効果量を推定
- より正確なサンプルサイズ計算

**戦略4: 実質的最小効果（SESOI）の設定**
- 理論的・実践的に意味のある最小効果を特定
- その検出に必要なサンプルサイズを計算

## VI. ソフトウェア・ツール

### G*Power

**特徴**
- 無料、最も広く使用される
- t検定、ANOVA、回帰、相関等をカバー
- 事前・事後・感度分析すべて可能

**ダウンロード**：https://www.psychologie.hhu.de/arbeitsgruppen/allgemeine-psychologie-und-arbeitspsychologie/gpower

**基本的使用手順**
1. Testファミリー選択（例：F tests）
2. 統計的検定選択（例：Linear multiple regression: Fixed model, R² deviation from zero）
3. Typeオプション選択（A priori: 必要サンプル計算）
4. パラメータ入力（効果量、α、検出力、予測変数数）
5. Calculate

### R/Pythonパッケージ

**R: pwr パッケージ**
```R
library(pwr)
# t検定
pwr.t.test(d = 0.5, sig.level = 0.05, power = 0.80, type = "two.sample")

# 相関
pwr.r.test(r = 0.30, sig.level = 0.05, power = 0.80)

# 回帰
pwr.f2.test(u = 8, f2 = 0.15, sig.level = 0.05, power = 0.80)
```

**Python: statsmodels**
```python
from statsmodels.stats.power import TTestIndPower, FTestAnovaPower
analysis = TTestIndPower()
n = analysis.solve_power(effect_size=0.5, alpha=0.05, power=0.80)
```
