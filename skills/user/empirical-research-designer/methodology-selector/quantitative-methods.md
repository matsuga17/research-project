# 量的研究方法論ガイド

## 概要

量的研究は、変数間の関係を数値データで検証し、一般化可能な知見を導出する。本ガイドは、サーベイ研究、準実験デザイン、統計分析の詳細を提供する。

---

## 1. サーベイ研究（Survey Research）

### 1.1 質問項目設計

**Churchill (1979) 尺度開発パラダイム**

**ステップ1: 測定すべきものの指定**
- 構成概念の明確な定義
- 理論的根拠

**ステップ2: 項目生成**
- 文献レビュー
- 専門家インタビュー
- 項目プール作成（目標の2-3倍）

**ステップ3: 測定尺度の精緻化**
- パイロットテスト（n=30-50）
- 項目分析、信頼性評価
- 不適切項目の削除

**ステップ4: 妥当性評価**
- 収束的妥当性：AVE ≥ 0.50
- 弁別的妥当性：√AVE > 構成概念間相関
- 確認的因子分析（CFA）

### 1.2 サンプリング戦略

**確率サンプリング**
- 単純無作為抽出
- 層化抽出（比例配分）
- クラスター抽出

**サンプルサイズ計算**（G*Power使用）
```
必要サンプルサイズ = (Z_α/2 + Z_β)² × 2σ² / δ²

標準設定:
α = 0.05, β = 0.20（検出力80%）
効果量: 小d=0.2, 中d=0.5, 大d=0.8
```

**実践的ルール**
- 回帰分析：変数数×15-20
- 因子分析：項目数×5-10、最低300
- SEM：最低200、推奨400+

### 1.3 回収率向上策

- 事前通知
- パーソナライズされた依頼
- 複数回のリマインダー
- 謝礼（金銭/結果報告書）
- 短縮版の提供

## 2. 準実験デザイン（Quasi-Experimental Design）

### 2.1 デザインタイプ（Shadish et al. 2002）

**非等質統制群デザイン**
- 処置群・統制群あり、ランダム割当なし
- 選択バイアスのリスク
- 対策：傾向スコアマッチング（PSM）

**中断時系列デザイン**
- 介入前後の時系列データ
- 歴史効果の統制
- 自己回帰統合移動平均（ARIMA）モデル

**回帰不連続デザイン（RDD）**
- カットオフ値による処置割当
- 因果推論の強い内的妥当性
- 局所的効果の推定

### 2.2 内的妥当性の脅威

- **選択バイアス**：PSM, IV法
- **歴史効果**：統制群、DiD
- **成熟効果**：縦断的統制
- **測定の反応性**：複数測定、盲検化

### 2.3 分析手法

**傾向スコアマッチング（PSM）**
```
1. 傾向スコア推定（ロジスティック回帰）
2. マッチング（最近傍、カーネル）
3. バランス検証（標準化差 < 0.1）
4. 処置効果推定（ATT, ATE）
```

**操作変数法（IV）**
```
必要条件:
- 操作変数と処置の相関（関連性）
- 操作変数と結果の直接的関連なし（外生性）
- 操作変数と結果の関係が処置を通じてのみ（除外制約）

検証: 弱操作変数検定（F > 10）
```

**差分の差分法（DiD）**
```
Y_it = α + β₁Treated_i + β₂Post_t + β₃(Treated × Post) + ε_it

β₃: 処置効果（DiD推定値）
並行トレンド仮定の検証必須
```

## 3. パネルデータ分析

### 3.1 モデル選択

**固定効果モデル（FE）**
- 個体固有効果を統制
- 時間不変の個体差を除去
- Within推定

**変量効果モデル（RE）**
- 個体効果を確率変数として扱う
- より効率的推定（FEより）
- 外生性仮定必要

**Hausman検定**
- FE vs RE選択
- p < 0.05 → FE推奨

### 3.2 動的パネルモデル

**System GMM（Blundell & Bond 1998）**
- 内生性、脱落変数バイアス対応
- 水準方程式+差分方程式
- Hansen J検定（過剰識別制約）
- Arellano-Bond検定（AR(2)なし）

## 4. 構成概念妥当性検証

### 4.1 信頼性（Reliability）

**Cronbach's α**
```
α = (k/(k-1)) × (1 - Σσ²_i / σ²_total)

基準: α ≥ 0.70（探索的）, α ≥ 0.80（確認的）
```

**合成信頼性（CR）**
```
CR = (Σλ)² / [(Σλ)² + Σθ]

基準: CR ≥ 0.70
```

### 4.2 収束的妥当性

**平均分散抽出（AVE）**
```
AVE = Σλ² / (Σλ² + Σθ)

基準: AVE ≥ 0.50
```

**因子負荷量**
```
基準: λ ≥ 0.50（推奨 λ ≥ 0.70）
```

### 4.3 弁別的妥当性

**Fornell-Larcker基準**
```
√AVE_i > r_ij (すべてのj≠iについて)
```

**HTMT比率**
```
HTMT < 0.85（保守的 < 0.90）
```

### 4.4 確認的因子分析（CFA）

**適合度指標**
```
χ²/df < 3（推奨 < 2）
CFI ≥ 0.95（許容 ≥ 0.90）
TLI ≥ 0.95（許容 ≥ 0.90）
RMSEA ≤ 0.06（許容 ≤ 0.08）
SRMR ≤ 0.08
```

## 5. 統計的仮定の検証

### 5.1 正規性

- **Shapiro-Wilk検定**：n < 2000
- **Kolmogorov-Smirnov検定**：n ≥ 2000
- **QQプロット**：視覚的確認

**対策**
- 変数変換（log, sqrt）
- 頑健推定（robust methods）
- ブートストラップ

### 5.2 等分散性

- **Levene検定**
- **Breusch-Pagan検定**（回帰）

**対策**
- 頑健標準誤差（robust SE）
- WLS（加重最小二乗法）

### 5.3 多重共線性

**VIF（分散拡大要因）**
```
VIF_i = 1 / (1 - R²_i)

基準: VIF < 10（保守的 < 5）
```

**対策**
- 変数削除
- 主成分回帰
- Ridge回帰

### 5.4 自己相関

**Durbin-Watson統計量**
```
DW ≈ 2: 自己相関なし
DW < 2: 正の自己相関
DW > 2: 負の自己相関
```

**対策**
- Newey-West標準誤差
- 動的モデル（ラグ変数追加）

## 6. 媒介・調整効果分析

### 6.1 媒介効果（Mediation）

**Baron & Kenny (1986) 手順**
1. X → Y（有意）
2. X → M（有意）
3. M → Y（X統制下で有意）
4. X → Y（M統制下で係数減少）

**Sobel検定**
```
z = a×b / √(a²s²_b + b²s²_a)
```

**Hayes PROCESS マクロ**
- ブートストラップ信頼区間
- 間接効果の検定
- 複数媒介モデル

### 6.2 調整効果（Moderation）

**交互作用項**
```
Y = β₀ + β₁X + β₂M + β₃(X×M) + ε

β₃: 調整効果
```

**単純傾斜分析**
- M高群でのXの効果
- M低群でのXの効果

**Johnson-Neyman法**
- 効果が有意な範囲の特定

## 7. 構造方程式モデリング（SEM）

### 7.1 モデル適合度

**絶対適合度指標**
- χ²統計量（非有意が望ましい）
- RMSEA ≤ 0.06
- SRMR ≤ 0.08

**相対適合度指標**
- CFI ≥ 0.95
- TLI ≥ 0.95

**簡約適合度指標**
- AIC, BIC（小さいほど良い）

### 7.2 モデル修正

**修正指標（Modification Indices）**
- MI > 3.84（p < 0.05）で改善可能

**注意点**
- 理論的根拠必須
- データドリブン修正の限界

## 8. 実践ガイド

### データクリーニング

1. **欠損値処理**
   - MCAR, MAR, MNAR診断
   - リストワイズ削除 vs 多重代入
   
2. **外れ値検出**
   - 箱ひげ図、Mahalanobis距離
   - Cook's距離（影響力診断）

3. **データ変換**
   - 中心化（centering）
   - 標準化（standardization）
   - 対数変換

### サンプルサイズ設計

**統計的検出力分析**（G*Power）
```
Input:
- 効果量（小/中/大）
- 有意水準 α = 0.05
- 検出力 1-β = 0.80

Output:
- 必要サンプルサイズ
```

### 結果報告

**記述統計**
- 平均、標準偏差、最小値、最大値
- 相関行列（Pearson r）

**推定結果**
- 係数、標準誤差、t値、p値
- R², 調整済みR²
- 適合度指標

**図表**
- 効果プロット
- 交互作用グラフ
- パス図（SEM）
