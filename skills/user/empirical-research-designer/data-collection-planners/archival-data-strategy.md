# アーカイブデータ戦略

## データソース

### 政府統計
経済センサス、工業統計、労働統計。信頼性高、無料。制約：集計データ、時間遅延。

### 企業データベース
Compustat, ORBIS, 東洋経済データバンク。財務・企業特性データ。制約：費用、アクセス制限。

### アーカイブ記録
図書館、公文書館。歴史的研究に不可欠。制約：整理されていない、欠損。

---

## データ品質評価

### 完全性
欠損値の割合、期間カバレッジ、変数利用可能性

### 信頼性
データ収集方法の文書化、測定誤差の程度、一貫性

### 妥当性
構成概念妥当性、測定尺度の適切性

---

## データアクセス手順

1. データソース特定
2. アクセス要件確認（申請、費用）
3. データ形式確認（API, CSV, Excel等）
4. サンプルデータ取得
5. 前処理計画作成
