# Tokyo-Cite クイックスタートガイド

## 概要

Tokyo-Citeは東京大学スタイルの引用管理を自動化するスキルです。

## 主要機能

1. **本文引用の自動変換**: （著者名, YYYY.M.D, ページ）形式に自動変換
2. **参考文献リストの自動生成**: 文献タイプ別に適切なフォーマット適用
3. **URL検証**: すべてのURLの接続確認と代替URL探索
4. **整合性チェック**: 本文引用と参考文献リストの完全一致確認

## クイックスタート

### ステップ1: 情報収集

```
タイプ: 雑誌論文
著者: 山田花子
タイトル: プライベートブランドの新展開
発行年月日: 2022.6.10
雑誌名: マーケティングジャーナル
巻号: 15(2)
ページ: pp.112-125
URL: https://www.marketing-journal.jp/article/12345
```

### ステップ2: 本文引用生成

**出力**:
```
（山田, 2022.6.10, pp.112-125）
```

### ステップ3: 参考文献生成

**出力**:
```
山田花子（2022）「プライベートブランドの新展開」『マーケティングジャーナル』15(2), pp.112-125.
https://www.marketing-journal.jp/article/12345 [アクセス日: 2025.11.22]
```

### ステップ4: URL検証

```
✅ URL検証: 成功
ステータスコード: 200
最終確認日: 2025.11.22
```

## 基本形式

### 本文引用

- **単著者**: （田中, 2023.4.15, p.42）
- **複数著者（2名）**: （田中・山田, 2023.4.15, p.42）
- **複数著者（3名以上）**: （田中ほか, 2023.4.15, p.42）
- **組織**: （経済産業省, 2023.4.15, p.57）

### 参考文献

#### 書籍
```
著者名（発行年）『書名』出版社, ページ数.
URL [アクセス日: YYYY.MM.DD]
```

#### 雑誌論文
```
著者名（発行年）「論文タイトル」『雑誌名』巻(号), pp.開始-終了.
URL [アクセス日: YYYY.MM.DD]
```

#### ウェブサイト
```
著者名/組織名（発行年月日）「ページタイトル」『ウェブサイト名』
URL [アクセス日: YYYY.MM.DD]
```

## 重要ルール

1. **引用長制限**: 直接引用は25語以下
2. **1ソース1引用**: 各情報源から1か所のみ引用
3. **URL必須**: すべての参考文献にURL記載（接続確認済み）
4. **パラフレーズ優先**: 直接引用は必要最小限

## 効率化効果

- 引用作成: **2-3分 → 30秒**（83-85%削減）
- 参考文献リスト: **30分 → 5分**（83%削減）
- URL検証: **1-2分/件 → 10秒/件**（92-95%削減）
