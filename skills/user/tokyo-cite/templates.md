# Tokyo-Cite 引用テンプレート集

## 本文引用テンプレート

### 1. 基本形式

#### 単著者
```
（著者姓, YYYY.M.D, p.XX）
```
**例**: （田中, 2023.4.15, p.42）

#### 複数著者（2名）
```
（著者姓・著者姓, YYYY.M.D, p.XX）
```
**例**: （田中・山田, 2023.4.15, p.42）

#### 複数著者（3名以上）
```
（筆頭著者姓ほか, YYYY.M.D, p.XX）
```
**例**: （田中ほか, 2023.4.15, p.42）

#### 組織・機関
```
（組織名, YYYY.M.D, p.XX）
```
**例**: （経済産業省, 2023.4.15, p.57）

#### オンライン資料（ページなし）
```
（著者姓, YYYY.M.D）
```
**例**: （田中, 2023.4.15）

#### 発行日不明
```
（著者姓, n.d., p.XX）
```
**例**: （田中, n.d., p.42）

#### 複数情報源の統合
```
（著者姓, YYYY.M.D; 著者姓, YYYY.M.D）
```
**例**: （田中, 2023.4.15; 山田, 2022.6.10）

#### 二次引用
```
（原著者姓, 原著発表年, 二次著者姓, 二次発表年, p.XX）
```
**例**: （Smith, 2018, 田中, 2023, p.25）

## 参考文献リストテンプレート

### 1. 書籍

#### 基本形式
```
著者姓名（発行年）『書名』出版社, ページ数.
URL [アクセス日: YYYY.MM.DD]
```

#### 記入例
```
【著者姓名】: 田中太郎
【発行年】: 2023
【書名】: デジタル時代のブランド戦略
【出版社】: 東京出版
【ページ数】: pp.42-45
【URL】: https://www.example.com/digital-brand
【アクセス日】: 2025.11.22

↓

田中太郎（2023）『デジタル時代のブランド戦略』東京出版, pp.42-45.
https://www.example.com/digital-brand [アクセス日: 2025.11.22]
```

#### 複数著者
```
著者姓名・著者姓名・著者姓名（発行年）『書名』出版社, ページ数.
URL [アクセス日: YYYY.MM.DD]
```

#### 翻訳書
```
著者姓名（訳者姓名訳）（発行年）『書名』出版社, ページ数.
URL [アクセス日: YYYY.MM.DD]
```

### 2. 雑誌論文

#### 基本形式
```
著者姓名（発行年）「論文タイトル」『雑誌名』巻(号), 開始ページ-終了ページ.
URL [アクセス日: YYYY.MM.DD]
```

#### 記入例
```
【著者姓名】: 山田花子
【発行年】: 2022
【論文タイトル】: プライベートブランドの新展開
【雑誌名】: マーケティングジャーナル
【巻】: 15
【号】: 2
【ページ】: 112-125
【URL】: https://www.marketing-journal.jp/article/12345
【アクセス日】: 2025.11.22

↓

山田花子（2022）「プライベートブランドの新展開」『マーケティングジャーナル』15(2), pp.112-125.
https://www.marketing-journal.jp/article/12345 [アクセス日: 2025.11.22]
```

#### DOI付き論文（推奨）
```
著者姓名（発行年）「論文タイトル」『雑誌名』巻(号), 開始ページ-終了ページ.
https://doi.org/XX.XXXX/XXXXXX [アクセス日: YYYY.MM.DD]
```

#### オンラインのみ（印刷版なし）
```
著者姓名（発行年）「論文タイトル」『雑誌名』オンライン版.
URL [アクセス日: YYYY.MM.DD]
```

### 4. 政府刊行物・報告書

#### 基本形式
```

URL [アクセス日: YYYY.MM.DD]
```

#### 記入例
```
【発行機関名】: 経済産業省
【発行年】: 2023
【報告書タイトル】: 令和5年版 小売業白書
【ページ数】: p.57
【URL】: https://www.meti.go.jp/report/retail2023
【アクセス日】: 2025.11.22

↓

経済産業省（2023）『令和5年版 小売業白書』経済産業省, p.57.
https://www.meti.go.jp/report/retail2023 [アクセス日: 2025.11.22]
```

#### 省庁報告書（著者明記）
```
著者姓名（発行年）『報告書タイトル』発行機関, ページ数.
URL [アクセス日: YYYY.MM.DD]
```

#### 国際機関
```
国際機関名（発行年）『報告書タイトル（英語）』国際機関名, ページ数.
URL [アクセス日: YYYY.MM.DD]
```

**例**:
```
World Bank（2023）『World Development Report 2023』World Bank, p.25.
https://www.worldbank.org/wdr2023 [アクセス日: 2025.11.22]
```

## 特殊ケーステンプレート

### URL接続不能時

#### Internet Archive使用
```
著者姓名/組織名（発行年月日）「タイトル」『サイト名/雑誌名』
オリジナルURL [Internet Archive: https://web.archive.org/web/YYYYMMDD/オリジナルURL] [アクセス日: YYYY.MM.DD]
```

#### 接続不能・代替なし
```
著者姓名/組織名（発行年月日）「タイトル」『サイト名/雑誌名』
[オリジナルURL - YYYY.MM.DD時点で接続不能。代替URL検索実施済み]
```

### DOI優先記載
```
著者姓名（発行年）「論文タイトル」『雑誌名』巻(号), 開始ページ-終了ページ.
https://doi.org/XX.XXXX/XXXXXX [アクセス日: YYYY.MM.DD]
```

### 複数URL併記
```
著者姓名/組織名（発行年月日）「タイトル」『サイト名/雑誌名』
一次URL [代替: 二次URL] [アクセス日: YYYY.MM.DD]
```
