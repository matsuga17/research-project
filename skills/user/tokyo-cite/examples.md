## 実践シナリオ別使用例

### シナリオ1: 学術論文の執筆

**状況**: 修士論文でプライベートブランド戦略について執筆中

**収集した情報源**:
1. 学術論文（雑誌）
2. 企業のIR資料
3. 政府統計報告書
4. ニュース記事

---

#### 例1-1: 学術論文の引用

**情報収集**:
```
タイプ: 雑誌論文
著者: 山田花子
タイトル: プライベートブランドの戦略的展開：小売業の競争優位構築
発行年月日: 2022.6.10
雑誌名: マーケティングジャーナル
巻号: 15(2)
ページ: pp.112-125
DOI: https://doi.org/10.1234/mj.2022.15.2.112
URL: https://www.marketing-journal.jp/article/12345
```

**本文での使用**:
```
プライベートブランド（以下、PB）は、小売業における重要な差別化戦略として位置づけられている。
山田（2022.6.10, p.115）は、PBの成功要因として「価格競争力」と「品質向上」の両立を挙げており、
特に日本市場では品質への期待が高いことを実証的に示している。
```

**参考文献リスト**:
```
山田花子（2022）「プライベートブランドの戦略的展開：小売業の競争優位構築」『マーケティングジャーナル』15(2), pp.112-125.
https://doi.org/10.1234/mj.2022.15.2.112 [アクセス日: 2025.11.22]
```

## ベストプラクティスまとめ

### データ収集フェーズ

1. **早期記録**: 資料発見時に即座に引用情報を記録
2. **完全情報**: すべての必須項目を初回から記録
3. **URL即時検証**: 資料収集直後にURL接続テスト
4. **DOI優先**: 学術論文は常にDOI確認・使用
5. **バックアップ**: Internet Archiveへの登録推奨

### 執筆フェーズ
