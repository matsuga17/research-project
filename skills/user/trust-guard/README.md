# TrustGuard - 情報信頼性評価・ハルシネーション防止システム

**バージョン**: 1.0.0  
**カテゴリ**: 学術研究支援、品質保証、事実検証  
**最終更新**: 2025-11-22

---

## 📌 概要

TrustGuardは、学術研究および情報分析における情報源の信頼性を多層的に検証し、ハルシネーション（事実に基づかない情報生成）を自動的に防止するスキルです。研究の学術的厳密性と再現可能性を担保し、週5-7時間の作業時間を削減します。

---

## ⚡ 主要機能

### 1. 多層検証フレームワーク（4段階）
- **存在検証**: URL接続確認、検索エンジン確認、アーカイブ確認
- **内容検証**: 引用と原文の完全一致確認（一致度95%以上）
- **整合性検証**: 複数情報源の交差検証、論理的整合性チェック
- **文脈検証**: 選択的引用検出、意図的歪曲の防止

### 2. 信頼性スコアリング（0-100点）
- 情報源タイプ別基礎スコア（学術論文95-100点 〜 検証不能0-29点）
- 加点要素: DOI、専門性、引用数、一次情報源など
- 減点要素: 利益相反、論理矛盾、古いデータなど

### 3. ハルシネーション検出
- **固有名詞検証**: 人名、組織名、地名の実在性確認
- **数値・統計検証**: 公式統計との照合、論理的妥当性チェック
- **引用存在性検証**: 実在しない情報源の検出
- **確証バイアス検出**: 一方的な情報選択の防止

### 4. 代替情報源自動探索
- 3段階検索: 学術情報源 → 公式情報源 → 高品質メディア
- 信頼性スコア付き推奨（上位3-5件）
- プライマリソースへの誘導

## 💡 使用例

### 例1: 学術論文の検証

**入力**:
```
以下の情報源を検証してください：

Smith, J. (2023). "The Impact of AI on Labor Markets"
Journal of Economic Studies, 85(4), pp.567-589.
https://doi.org/10.1234/jes.2023.85.4.567

引用箇所: "AI導入により製造業の雇用は2030年までに15%削減される"
```

**出力**:
- 信頼性スコア: 88点（高い信頼性）
- 多層検証結果: すべて合格
- 推奨アクション: 一般的な学術利用に適格
- 改善提案: 前提条件の明示、論争的論点の追記

### 例2: ニュース記事の検証

**入力**:
```
日本経済新聞（2024.3.15）「AI導入で企業の生産性30%向上」
から引用したいのですが、信頼性を確認してください。
```

**出力**:
- 信頼性スコア: 62点（中程度の信頼性）
- 問題点: サンプルサイズ小、限定条件の省略
- 推奨: より信頼性の高い代替ソース3件を提示
- 改善提案: 「約」の追加、限定条件の明示

## 📚 検証プロセス

### 標準モード（15-20分）
1. **Stage 1: 初期スキャン**（1-2分）
2. **Stage 2: 多層検証**（3-5分）
3. **Stage 3: 詳細分析**（5-10分）
4. **Stage 4: レポート生成**（2-3分）

### クイック検証モード（2-3分）
- URL接続確認
- 基礎スコア算出
- 明らかな問題の指摘

---

## ✅ ベストプラクティス

### Do's（推奨）
- ✅ 検証は早い段階で実施
- ✅ 複数の独立情報源を使用
- ✅ 一次情報源を優先
- ✅ 限定条件を明示
- ✅ 対立する見解も提示

### Don'ts（禁止）
- ❌ 未検証情報の使用
- ❌ 単一情報源への依存
- ❌ 文脈を無視した引用
- ❌ 信頼性スコア60点未満の単独使用
- ❌ 確証バイアスの放置

## 🔧 トラブルシューティング

### よくある問題

**URL接続不能（404）**
→ Internet Archive確認、Google検索で代替URL探索、DOI経由アクセス

**引用内容が原文で見つからない**
→ キーワード部分一致検索、類似表現検索、一次情報源に直接あたる

**矛盾する情報源が存在**
→ 調査時期・方法・サンプルを確認、最新かつ最も信頼性の高い情報源を優先

**信頼性スコアが低い**
→ 代替情報源探索機能使用、より信頼性の高いソース探索
