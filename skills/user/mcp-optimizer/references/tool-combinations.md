# ツール組み合わせガイド

## 1. 研究タスク向け組み合わせ

### 1.1 文献調査

**ツールセット**:
```
Semantic Scholar + Google Scholar + Desktop Commander
```

**最適化パターン**:
```
1. Semantic Scholar で学術論文検索
   → /home/claude/papers/semantic_scholar.csv

2. Google Scholar で補完検索
   → /home/claude/papers/google_scholar.csv

3. Desktop Commander で結果統合・重複除去
   → /home/claude/papers/merged.csv

報告: 「統合後{N}件。重複除去{M}件」
```

**効率化ポイント**:
- 各検索結果は個別ファイルに保存
- 統合処理後に初めて内容を確認
- コンテキスト消費: 約80%削減

### 1.2 企業データ収集

**ツールセット**:
```
Web Search + Exa + Desktop Commander + (SEC EDGAR if applicable)
```

**最適化パターン**:
```
1. Web Search で基本情報取得
   → /home/claude/companies/{name}/basic.json

2. Exa で最新ニュース・動向
   → /home/claude/companies/{name}/news.json

3. SEC EDGAR で財務情報（米国企業の場合）
   → /home/claude/companies/{name}/sec.json

4. Desktop Commander で統合
   → /home/claude/companies/{name}/profile.json

報告: 「{企業名}のデータ収集完了。項目カバレッジ{%}」
```

### 1.3 市場調査

**ツールセット**:
```
Web Search + Exa + Semantic Scholar + Desktop Commander
```

**最適化パターン**:
```
Phase 1: 情報収集（並列実行）
├── Web Search: 業界レポート → /home/claude/market/reports.md
├── Exa: 最新トレンド → /home/claude/market/trends.md
└── Semantic Scholar: 学術分析 → /home/claude/market/academic.md

Phase 2: 統合分析
└── Desktop Commander: 統合 → /home/claude/market/analysis.md

報告: 「3ソースから情報統合。主要インサイト: {points}」
```

## 5. トラブルシューティング

### 問題1: ファイル保存の失敗

**原因**: パスの誤り、権限問題
**対処**: 
```
1. /home/claude/ 配下を使用
2. ディレクトリを先に作成
3. エラー時は /home/claude/temp/ に一時保存
```

### 問題2: ツール間のデータ形式不一致

**原因**: JSONとCSVの混在
**対処**:
```
1. 統合前に形式を統一
2. JSONを基本とし、分析時にCSV変換
3. 変換スクリプトを用意
```

### 問題3: 大量データによるタイムアウト

**原因**: 一度に処理するデータ量が多すぎる
**対処**:
```
1. バッチサイズを制限（10-20件/バッチ）
2. 中間保存を頻繁に
3. タイムアウト時は中間結果から再開
```
