# US Corporate Analytics - 米国企業財務・ガバナンス分析スキル

**バージョン**: 1.0.0  
**作成日**: 2025-11-02  
**対応分野**: 経営戦略、コーポレートファイナンス、コーポレートガバナンス、実証研究

---

## 【即答】このスキルでできること

1. **SEC EDGAR統合分析**：8,000社以上の米国上場企業の財務・ガバナンスデータを自動取得・分析
2. **包括的財務分析**：30種類以上の財務比率、トレンド分析、業界比較
3. **ガバナンス分析**：取締役会構成、役員報酬、株主構造の詳細分析
4. **マクロ経済統合**：World Bank・IMFデータと企業データの統合分析
5. **学術研究対応**：AEA準拠のデータ品質、Publication-gradeの分析レポート

---

## スキル概要

このスキルは、米国企業の財務パフォーマンスとコーポレートガバナンス体制を包括的に分析するための統合システムです。SEC EDGAR API、World Bank Open Data、IMF Dataの3つの主要データソースを統合し、学術研究からビジネス実務まで幅広いニーズに対応します。

### 主要機能

#### 1. データ取得モジュール
- **SEC EDGAR API統合**
  - 10-K（年次報告書）の自動取得
  - 10-Q（四半期報告書）の時系列取得
  - 8-K（重要事象報告）のイベント抽出
  - DEF 14A（委任状説明書）からのガバナンスデータ抽出
  - Form 4（内部者取引）の取引分析

- **World Bank Data統合**
  - マクロ経済指標（GDP、インフレ率、失業率）
  - ビジネス環境指標（Doing Business Index）
  - 制度的環境指標（法の支配、汚職度）

- **IMF Data統合**
  - 国際金融統計
  - 為替レート（実効為替レート）
  - 国際収支データ

#### 2. 財務分析モジュール
- **収益性分析**：ROA、ROE、営業利益率、純利益率
- **流動性分析**：流動比率、当座比率、現金比率
- **レバレッジ分析**：負債比率、D/E比率、インタレストカバレッジ
- **効率性分析**：総資産回転率、売上債権回転率、棚卸資産回転率
- **成長性分析**：売上成長率、利益成長率、資産成長率
- **市場価値分析**：EPS、PER、PBR、配当利回り
- **キャッシュフロー分析**：営業CF、投資CF、財務CF、FCF

#### 3. ガバナンス分析モジュール
- **取締役会分析**
  - 取締役会規模と構成
  - 独立取締役比率
  - 取締役の多様性（性別、年齢、国籍）
  - 取締役会の開催頻度

- **役員報酬分析**
  - CEO報酬の構成（基本給、ボーナス、株式報酬）
  - Pay-for-Performance分析
  - 業界比較
  - 報酬格差分析

- **株主構造分析**
  - 大株主の保有比率
  - 機関投資家の保有状況
  - 内部者保有比率
  - 株主の集中度（HHI指数）

#### 4. 比較分析モジュール
- **時系列分析**：企業の経年変化トレンド
- **業界比較**：SICコードベースの同業他社比較
- **ピア分析**：選択企業群との詳細比較
- **パフォーマンスベンチマーク**：業界トップ企業との比較

#### 5. レポート生成モジュール
- **エグゼクティブサマリー**：経営層向け1ページサマリー
- **詳細財務分析レポート**：30ページ以上の包括的レポート
- **ガバナンス評価レポート**：取締役会・報酬・株主構造の評価
- **業界分析レポート**：競合環境と市場ポジション分析
- **学術研究レポート**：査読論文投稿用の統計分析レポート

## データソースの詳細

### 1. SEC EDGAR API

**APIキー**: e1bfb8c9d7c5f3538093cb5f1dc9327b71279ad3854418d967ba37dad594f0b8

**エンドポイント**: https://www.sec.gov/cgi-bin/browse-edgar

**レート制限**: 10リクエスト/秒

**対応書類**:

| 書類タイプ | 内容 | 更新頻度 | データ範囲 |
|-----------|------|---------|----------|
| 10-K | 年次報告書（包括的財務データ） | 年次 | 1993年〜現在 |
| 10-Q | 四半期報告書 | 四半期 | 1993年〜現在 |
| 8-K | 重要事象報告（M&A、CEO交代等） | イベント発生時 | 1993年〜現在 |
| DEF 14A | 委任状説明書（役員報酬、ガバナンス） | 年次 | 1993年〜現在 |
| Form 4 | 内部者取引報告 | 取引発生後2営業日 | 1993年〜現在 |
| S-1 | IPO登録届出書 | IPO時 | 1993年〜現在 |

**データ品質保証**:
- XBRL形式での構造化データ（2009年以降）
- Benford's Law準拠の異常値検出
- クロスチェック機能（複数書類での整合性確認）

### 2. World Bank Open Data

**APIエンドポイント**: https://api.worldbank.org/v2/

**認証**: 不要（オープンアクセス）

**主要指標**:

**マクロ経済指標**:
- `NY.GDP.MKTP.CD`: GDP（名目、米ドル）
- `NY.GDP.MKTP.KD.ZG`: GDP成長率（実質）
- `FP.CPI.TOTL.ZG`: インフレ率（CPI）
- `SL.UEM.TOTL.ZS`: 失業率

**ビジネス環境指標**:
- `IC.BUS.EASE.XQ`: Doing Business Index
- `IC.REG.DURS`: 起業にかかる日数
- `IC.FRM.TIME`: 契約執行にかかる日数

**制度的環境指標**:
- `CC.EST`: 汚職統制指標
- `RL.EST`: 法の支配指標
- `RQ.EST`: 規制品質指標

**データ範囲**: 1960年代〜現在（指標により異なる）

### 3. IMF Data

**APIエンドポイント**: https://www.imf.org/external/datamapper/api/v1/

**認証**: 不要（オープンアクセス）

**主要データセット**:
- **国際金融統計（IFS）**: 為替レート、金利、マネーサプライ
- **政府財政統計（GFS）**: 政府支出、税収、債務
- **国際収支統計（BOP）**: 貿易収支、資本フロー、FDI

**データ範囲**: 1950年代〜現在

## 使用方法

### 基本的な使い方

#### 1. 単一企業の財務分析

```
「Appleの過去5年間の財務分析を実施してください」
```

**Claudeの動作**:
1. SEC EDGARからApple（CIK: 0000320193）の10-Kを取得
2. 2019-2023年の財務データを抽出
3. 30種類以上の財務比率を計算
4. 時系列トレンドを可視化
5. 業界平均と比較
6. 包括的なレポートを生成

**出力形式**: 
- PDF形式の詳細レポート（30ページ）
- Excel形式の財務データ（生データ + 計算済み比率）
- 可視化チャート（PNG形式）

---

#### 2. 複数企業の比較分析

```
「Apple、Microsoft、Googleの財務パフォーマンスを比較してください。
重点項目：収益性、成長性、キャッシュフロー」
```

**Claudeの動作**:
1. 3社の10-Kデータを並行取得
2. 指定された重点項目の比率を計算
3. 相対的なパフォーマンスを可視化
4. 強み・弱みの分析
5. 比較レポートを生成

**出力形式**:
- 比較表（Excel）
- レーダーチャート（視覚的比較）
- ランキング表
- 詳細分析レポート（PDF）

---

#### 3. ガバナンス分析

```
「Teslaの取締役会構成と役員報酬を分析してください」
```

**Claudeの動作**:
1. DEF 14Aから取締役会データを抽出
2. 取締役の経歴、独立性を分析
3. 役員報酬の詳細を取得
4. 業界平均と比較
5. ガバナンス評価レポートを生成

**出力項目**:
- 取締役会の規模・構成
- 独立取締役比率
- 多様性指標（性別、年齢、専門性）
- CEO報酬の構成（基本給、ボーナス、株式報酬）
- Pay-for-Performance分析
- 業界比較

---

#### 4. マクロ経済環境との統合分析

```
「Amazonの業績とマクロ経済指標（GDP成長率、インフレ率）の
関係を分析してください」
```

**Claudeの動作**:
1. SEC EDGARからAmazonの財務データ取得
2. World Bankからマクロ経済指標取得
3. 相関分析の実施
4. 回帰分析（業績 ← マクロ指標）
5. 経済感応度レポートを生成

**統計手法**:
- Pearson相関係数
- 線形回帰分析
- 時系列分析（VAR/VECM）
- グレンジャー因果性検定

---

#### 5. イベント分析（8-K活用）

```
「過去3年間のM&A発表が株価に与えた影響を分析してください。
対象企業：S&P 500構成企業」
```

**Claudeの動作**:
1. 8-Kから「Item 2.01（重要な資産の取得・処分）」を抽出
2. M&A発表日を特定
3. CRSP株価データと統合（別途必要）
4. イベントスタディ分析（-5日〜+5日）
5. 累積異常リターン（CAR）を計算

**出力**:
- M&Aイベントリスト
- 平均CAR
- 成功/失敗の要因分析
- イベントスタディレポート

---

### 高度な使用例

#### 6. 学術研究向けデータセット構築

```
「製造業（SIC 20-39）の企業について、
2010-2023年の財務・ガバナンスデータセットを構築してください。
必要変数：ROA、ROE、レバレッジ、取締役会規模、CEO報酬」
```

**Claudeの動作**:
1. SICコード20-39の企業リストを取得
2. 各企業の10-K、DEF 14Aを一括取得
3. 指定された変数を抽出・計算
4. パネルデータ構造（企業-年次）で整理
5. 欠損値処理（補間 or 除外）
6. Benford's Law品質チェック
7. AEA準拠のデータ系譜を記録

**出力形式**:
- CSV形式（Stata/R/Pythonで直接利用可能）
- データ辞書（変数定義、単位、ソース）
- README.md（再現手順）
- 品質チェックレポート

---

#### 7. 業界全体の分析

```
「医薬品業界（SIC 2834）の財務トレンドを分析してください。
期間：2015-2023年
重点：R&D投資、収益性、成長性」
```

**Claudeの動作**:
1. SIC 2834企業の全リストを取得（約200社）
2. 各社の10-Kから財務データ取得
3. 業界平均・中央値の計算
4. トップ/ボトム企業の特定
5. R&D集約度の分析
6. 業界レポート生成

**出力**:
- 業界平均財務指標の時系列
- R&D投資と収益性の相関分析
- トップ10企業ランキング
- 業界構造分析（集中度、競争環境）

---

#### 8. ガバナンス改革の影響分析

```
「2018-2023年にCEOが交代した企業について、
交代前後の業績変化を分析してください」
```

**Claudeの動作**:
1. 8-K「Item 5.02（役員の異動）」からCEO交代を抽出
2. 交代前2年・後2年の財務データを取得
3. 差分の差分法（DID）で因果効果を推定
4. 外部登用 vs 内部昇格の比較
5. 統計的検定（t検定、回帰分析）

**出力**:
- CEO交代イベントリスト
- 業績変化の記述統計
- DID推定結果
- 外部/内部登用の比較表
- 学術論文形式の分析レポート

| スクリプト名 | 機能 | 使用例 |
|-------------|------|--------|
