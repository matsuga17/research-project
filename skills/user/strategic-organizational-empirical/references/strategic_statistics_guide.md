## 序章：統計的推論の認識論的基盤

統計手法は、単なる数学的技術ではなく、理論と証拠の間の認識論的橋渡しである。戦略・組織研究における統計的分析は、以下の三層構造として理解される：

**第一層：記述的推論** - 現象の体系的記述と要約  
**第二層：関係的推論** - 変数間の関連性の探索  
**第三層：因果的推論** - メカニズムと因果効果の解明

本ガイドは、各統計手法の数学的基盤だけでなく、理論的意味論、適用条件、戦略・組織研究における解釈方法を包括的に解説する。

## I. 多変量解析手法の選択基準

### 1.1 研究目的による分類

戦略・組織研究における統計手法は、研究目的により以下のように分類される：

**【目的1】変数間の関係性探索**
- 相関分析、回帰分析
- 構造方程式モデリング（SEM）
- パス解析

**【目的2】潜在変数の抽出**
- 因子分析（探索的・確認的）
- 主成分分析（PCA）
- 潜在クラス分析

**【目的3】グループ分類**
- クラスター分析
- 判別分析
- 多項ロジット分析

**【目的4】因果効果の推定**
- 回帰不連続デザイン（RDD）
- 差分の差分法（DiD）
- 操作変数法（IV）
- 傾向スコアマッチング（PSM）

### 1.2 データ特性による選択基準

**データ構造による分類**：

| データ型 | 推奨手法 | 理論的含意 |
|---------|---------|----------|
| 横断面データ | OLS回帰、SEM | 相関関係の探索、理論検証 |
| 時系列データ | ARIMA、VAR | 動態プロセスの解明 |
| パネルデータ | 固定効果、変量効果 | 観察されない異質性の統制 |
| 質的変数（二値） | ロジット、プロビット | 離散的選択の分析 |
| 質的変数（多値） | 多項ロジット、順序ロジット | 複数選択肢の分析 |
| カウントデータ | ポアソン、負の二項 | 頻度・発生数の分析 |

**サンプルサイズによる制約**：

```
n < 100:    非パラメトリック手法、ブートストラップ
100 ≤ n < 200:  基本的回帰分析、単純なSEM
200 ≤ n < 500:  複雑なSEM、多層モデル
n ≥ 500:    高度なモデリング、交互作用の精密検証
```

### 1.3 理論的前提による選択

**資源ベース理論（RBV）**：
- 推奨手法: 階層的回帰分析、パネル固定効果
- 理由: 企業固有の資源異質性を統制する必要

**取引コスト経済学（TCE）**：
- 推奨手法: ロジット分析、生存分析
- 理由: 統治構造の離散的選択を分析

**制度理論**：
- 推奨手法: イベントヒストリー分析、ネットワーク分析
- 理由: 制度的圧力の時間的・社会的拡散を捕捉

**動的能力論**：
- 推奨手法: 成長曲線モデル、動的パネル分析
- 理由: 能力の進化的プロセスを時系列で追跡

## II. 主要統計手法の理論的解釈

### 2.1 回帰分析（Regression Analysis）

#### 理論的基盤

回帰分析は、Galton (1886)に起源を持ち、独立変数が従属変数に与える影響を定量化する最も基本的な手法である。戦略研究では、理論的予測（仮説）を統計的検証に変換する中核的ツールとなる。

**数学的表現**：
```
Y = β₀ + β₁X₁ + β₂X₂ + ... + βₖXₖ + ε
```

**理論的解釈**：
- **β₀（切片）**: 全ての独立変数がゼロの時の従属変数の期待値
- **βᵢ（偏回帰係数）**: 他の変数を統制した時のXᵢの限界効果
- **R²（決定係数）**: モデルが説明する分散の割合（理論の説明力）
- **ε（誤差項）**: 観察されない要因、測定誤差、確率的変動

#### 戦略研究における応用

**例1: RBVの検証**

```
企業業績 = β₀ + β₁(R&D投資) + β₂(特許数) + β₃(企業規模) + ε
```

**解釈**：
- β₁が有意に正 → R&D投資（能力構築）が業績向上に寄与
- β₂が有意に正 → 知的財産（模倣困難性）が業績向上に寄与
- R² = 0.35 → モデルは業績分散の35%を説明（残り65%は他要因）

**注意点**：
- 因果関係ではなく相関関係を示す（内生性の問題）
- 省略変数バイアス（重要な変数の欠落）に注意
- 多重共線性（独立変数間の高相関）の診断必須

#### 発展的手法

**1. 階層的回帰分析（Hierarchical Regression）**

理論的予測を段階的に検証する手法：

```
Model 1（統制変数のみ）: Y = β₀ + β₁(Controls) + ε₁
Model 2（独立変数追加）:  Y = β₀ + β₁(Controls) + β₂(IV) + ε₂
Model 3（交互作用追加）:  Y = β₀ + β₁(Controls) + β₂(IV) + β₃(IV×Moderator) + ε₃
```

**ΔR²の解釈**：各段階での説明力の増分 = 理論的予測の追加的貢献

**2. 媒介分析（Mediation Analysis）**

因果メカニズムを検証する手法（Baron & Kenny, 1986; Hayes, 2013）：

```
Step 1:  Y = c·X + ε         (総効果)
Step 2:  M = a·X + ε         (X→M パス)
Step 3:  Y = c'·X + b·M + ε  (直接効果と媒介効果)

間接効果 = a × b
総効果 = c = c' + a×b
```

**戦略研究の例**：
- X: 動的能力
- M: 戦略的柔軟性（メカニズム）
- Y: 企業業績

**3. 調整分析（Moderation Analysis）**

理論的境界条件を検証する手法（Aiken & West, 1991）：

```
Y = β₀ + β₁(IV) + β₂(Moderator) + β₃(IV × Moderator) + ε
```

**解釈**：
- β₃が有意 → Moderatorが IV→Y の関係を強化/弱化
- Simple Slope分析：Moderatorの異なる水準でのIVの効果を算出

### 2.2 因子分析（Factor Analysis）

#### 理論的基盤

因子分析は、Spearman (1904)の「一般知能（g factor）」理論に起源を持つ。観察される多数の変数（指標）を、少数の潜在的構成概念（因子）で説明する手法である。

**基本的仮定**：
- 観測変数は潜在因子の線形結合として表現される
- 因子は直接観測できないが、複数の指標を通じて間接的に測定可能
- 測定誤差は各観測変数に独立に作用する

**数学的表現**：
```
Xᵢ = λᵢ₁F₁ + λᵢ₂F₂ + ... + λᵢₘFₘ + εᵢ

Xᵢ: i番目の観測変数
Fⱼ: j番目の共通因子
λᵢⱼ: 因子負荷量（Xᵢが Fⱼ に依存する度合い）
εᵢ: 独自因子（測定誤差）
```

#### 探索的因子分析（EFA）vs 確認的因子分析（CFA）

**探索的因子分析（EFA）**：
- **目的**: 因子構造の発見（理論生成）
- **適用場面**: 尺度開発の初期段階、理論的に未確立の領域
- **特徴**: 因子数とパターンはデータから導出
- **制約**: 事前の理論的制約なし

**確認的因子分析（CFA）**：
- **目的**: 理論的因子構造の検証（理論検証）
- **適用場面**: 確立された測定尺度の妥当性確認
- **特徴**: 因子数と因子負荷パターンを事前に指定
- **利点**: モデル適合度による厳密な評価

#### 戦略研究における応用

**例: 組織能力の多次元構造**

**理論的前提**：
組織能力は、以下の3つの下位次元から構成される（Winter, 2003）：
1. 学習能力（Learning Capability）
2. 統合能力（Integration Capability）
3. 再構成能力（Reconfiguration Capability）

**測定項目**（各次元3項目、計9項目）：

```
学習能力:
  LC1: R&D投資額
  LC2: 従業員訓練時間
  LC3: 外部知識吸収能力

統合能力:
  IC1: 部門間協働頻度
  IC2: 情報システム統合度
  IC3: クロスファンクショナルチーム数

再構成能力:
  RC1: 組織構造変革頻度
  RC2: プロセス革新件数
  RC3: 戦略的柔軟性指標
```

**CFA結果の解釈**：

```
因子負荷量:
  LC1 ← 学習能力: 0.82***
  LC2 ← 学習能力: 0.75***
  LC3 ← 学習能力: 0.68***
  
モデル適合度:
  CFI = 0.95 (良好)
  RMSEA = 0.06 (許容範囲)
  
因子間相関:
  学習能力 ↔ 統合能力: 0.54
  学習能力 ↔ 再構成能力: 0.48
  統合能力 ↔ 再構成能力: 0.61
```

**理論的含意**：
- 3つの下位次元は区別可能（弁別的妥当性）
- しかし中程度の相関（0.48-0.61）を持つ → 統合的構成概念の証拠
- これは、Winter (2003)の理論的予測と一致

#### 因子回転の理論的意味

**直交回転（Varimax）**：
- 因子間を無相関に制約
- 理論的含意: 構成概念が完全に独立

**斜交回転（Promax, Oblimin）**：
- 因子間の相関を許容
- 理論的含意: 構成概念が概念的に関連
- 戦略研究では通常こちらを選択（現実的）

### 2.3 構造方程式モデリング（SEM）

#### 理論的基盤

SEM（Structural Equation Modeling）は、因子分析と回帰分析を統合した手法である（Jöreskog, 1970; Bollen, 1989）。潜在変数間の因果関係を、測定誤差を考慮しながら検証できる。

**SEMの2つの構成要素**：

**1. 測定モデル（Measurement Model）**：
- 観測変数と潜在変数の関係を定義
- 因子分析に相当

**2. 構造モデル（Structural Model）**：
- 潜在変数間の因果関係を定義
- パス解析・回帰分析に相当

**数学的表現**：

```
測定モデル:
  X = Λₓξ + δ  (外生変数の測定)
  Y = Λᵧη + ε  (内生変数の測定)

構造モデル:
  η = Βη + Γξ + ζ

記号:
  ξ (ksi): 外生潜在変数
  η (eta): 内生潜在変数
  X, Y: 観測変数
  Λ (lambda): 因子負荷行列
  Β (beta): 内生変数間の係数
  Γ (gamma): 外生→内生の係数
  δ, ε, ζ: 誤差項
```

#### 戦略研究における応用

**例: RBVの統合的検証**

**理論モデル**：
```
資源蓄積 → 組織能力 → 競争優位 → 企業業績
    ↓           ↓
  (測定誤差)  (測定誤差)
```

**測定モデル**：
- 資源蓄積: 物的資源、人的資源、組織資源（各3項目）
- 組織能力: 学習能力、統合能力、再構成能力（各3項目）
- 競争優位: コスト優位、差別化優位（各3項目）
- 企業業績: ROA、市場シェア、成長率（各1項目）

**構造モデルの仮説**：
- H1: 資源蓄積 → 組織能力（+）
- H2: 組織能力 → 競争優位（+）
- H3: 競争優位 → 企業業績（+）

**結果の解釈**：

```
標準化パス係数:
  資源蓄積 → 組織能力: 0.68*** (R² = 0.46)
  組織能力 → 競争優位: 0.72*** (R² = 0.52)
  競争優位 → 企業業績: 0.55*** (R² = 0.30)

モデル適合度:
  χ²/df = 2.3 (良好、基準 <3.0)
  CFI = 0.94 (良好、基準 ≥0.90)
  RMSEA = 0.07 (許容、基準 <0.08)
  SRMR = 0.05 (優良、基準 <0.08)

総合判定: モデル適合度は許容可能
```

**理論的含意**：
- RBVの因果連鎖（資源→能力→優位→業績）が実証的に支持された
- 各段階での説明力（R²）は中程度〜高い
- しかし業績のR²=0.30 → 他の要因（環境、運等）も重要

#### SEMの高度な応用

**1. 媒介効果のSEM検証**

```
直接効果: 資源蓄積 → 企業業績 (c')
間接効果: 資源蓄積 → 組織能力 → 企業業績 (a×b)

ブートストラップ信頼区間で間接効果の有意性を検証
```

**2. 多母集団分析（Multi-Group Analysis）**

異なる産業、国、時期でモデルが同じか検証：

```
H: モデル構造は産業間で不変（Structural Invariance）

検証:
  1. 配置不変性（Configural Invariance）
  2. 測定不変性（Metric Invariance）
  3. 構造不変性（Structural Invariance）
```

**3. 縦断的SEM（Longitudinal SEM）**

時間を通じた因果関係の検証：

```
クロスラグモデル:
  能力(t) → 業績(t+1)
  業績(t) → 能力(t+1)

逆因果の検証が可能
```

### 2.4 パネルデータ分析

#### 理論的基盤

パネルデータは、同一対象（企業、個人等）を時系列で追跡したデータである。横断面データと時系列データの両方の利点を持つ。

**パネルデータの利点**：
1. **観察されない異質性の統制**: 企業固有の時不変要因（企業文化、創業者の資質等）を統制
2. **因果推論の強化**: 時間的先行性により因果方向を明確化
3. **動態プロセスの解明**: 変化のメカニズムを追跡

#### 固定効果モデル vs 変量効果モデル

**固定効果モデル（Fixed Effects, FE）**：

```
Yᵢₜ = β₀ + β₁Xᵢₜ + αᵢ + εᵢₜ

αᵢ: 企業i固有の時不変効果（固定効果）
```

**理論的解釈**：
- 各企業は固有の「切片」を持つ
- αᵢは説明変数と相関していてもよい
- Within変動（企業内の時系列変化）のみを利用

**適用場面**：
- 観察されない企業固有要因が重要な場合
- 例: 企業文化、地理的立地、創業者の資質

**変量効果モデル（Random Effects, RE）**：

```
Yᵢₜ = β₀ + β₁Xᵢₜ + (αᵢ + εᵢₜ)

αᵢ ~ N(0, σ²α): ランダムな企業効果
```

**理論的解釈**：
- 企業効果は母集団からのランダム抽出
- αᵢは説明変数と無相関を仮定
- Between変動とWithin変動の両方を利用

**適用場面**：
- 企業固有要因が説明変数と無関係な場合
- サンプルが母集団の代表的抽出の場合

#### モデル選択：Hausman検定

```
H₀: Cov(αᵢ, Xᵢₜ) = 0 （変量効果が適切）
H₁: Cov(αᵢ, Xᵢₜ) ≠ 0 （固定効果が適切）

p < 0.05 → 固定効果を選択
p ≥ 0.05 → 変量効果を選択
```

**戦略研究の実践的指針**：
- 迷ったら固定効果を選択（保守的）
- 企業固有要因が理論的に重要なら固定効果

#### 戦略研究における応用

**例: 動的能力と企業業績の因果関係**

**研究設定**：
- サンプル: 日本製造業200社
- 期間: 2015-2024年（10年間）
- N = 2000 (200企業 × 10年)

**モデル**：

```
業績ᵢₜ = β₀ + β₁(動的能力ᵢₜ) + β₂(企業規模ᵢₜ) + αᵢ + γₜ + εᵢₜ

αᵢ: 企業固定効果
γₜ: 年次固定効果（マクロ経済ショックを統制）
```

**結果**：

```
固定効果モデル:
  動的能力: β₁ = 0.34*** (SE = 0.08, p < 0.001)
  企業規模:  β₂ = 0.12*   (SE = 0.05, p < 0.05)
  
  R²（Within） = 0.28
  企業固定効果のF検定: F = 15.4*** → 固定効果が有意

Hausman検定:
  χ²(2) = 28.5, p < 0.001 → 固定効果が適切
```

**理論的解釈**：
- 企業内の動的能力の変化が、業績の変化に有意に寄与（β₁ = 0.34）
- 企業固有の時不変要因（創業者の資質、立地等）を統制した上での効果
- これは、動的能力が業績向上に因果的に寄与するという Teece et al. (1997)の理論を支持

#### 動的パネルモデル

過去の従属変数が現在の従属変数に影響する場合：

```
Yᵢₜ = ρYᵢₜ₋₁ + β₁Xᵢₜ + αᵢ + εᵢₜ

問題: Yᵢₜ₋₁ と αᵢ が相関 → OLS/FEは不一致推定量
解決: Arellano-Bond GMM推定（操作変数法）
```

**戦略研究の例**：
- 業績の持続性（慣性）を考慮
- 能力蓄積の累積的プロセスを分析

### 2.5 離散選択モデル

#### 理論的基盤

戦略研究では、連続的な業績だけでなく、離散的な選択も分析対象となる：
- 市場参入/撤退の選択
- 統治構造の選択（内製 vs 外注 vs 合弁）
- 戦略タイプの選択（コストリーダーシップ vs 差別化）

これらは通常の回帰分析では扱えず、離散選択モデルが必要。

#### ロジスティック回帰（Logistic Regression）

**二値選択の場合**（例: 市場参入 Yes/No）：

```
P(Y=1|X) = 1 / (1 + exp(-(β₀ + β₁X₁ + ... + βₖXₖ)))

対数オッズ（ロジット変換）:
  log(p/(1-p)) = β₀ + β₁X₁ + ... + βₖXₖ
```

**係数の解釈**：
- βᵢ: Xᵢが1単位増加した時の対数オッズの変化
- exp(βᵢ): オッズ比（OR, Odds Ratio）

**戦略研究の例**：

```
研究問題: どのような企業特性が国際市場参入を促進するか？

モデル:
  P(参入=1) = f(β₀ + β₁(企業規模) + β₂(国際経験) + β₃(製品差別化))

結果:
  企業規模: β₁ = 0.85*** → OR = exp(0.85) = 2.34
  
解釈:
  企業規模が1単位（例: 従業員1000人）増加すると、
  参入オッズが2.34倍になる
```

#### 多項ロジットモデル（Multinomial Logit）

3つ以上の選択肢がある場合（例: 統治構造の選択）：

```
選択肢: (1) 内製、(2) 外注、(3) 合弁

基準カテゴリーを(1)内製とすると:

P(Y=2) / P(Y=1) = exp(β₂₀ + β₂₁X₁ + ... + β₂ₖXₖ)
P(Y=3) / P(Y=1) = exp(β₃₀ + β₃₁X₁ + ... + β₃ₖXₖ)
```

**理論的応用（TCE）**：

```
変数:
  X₁: 資産特殊性（高いほど内製化）
  X₂: 取引頻度（高いほど内製化）
  X₃: 不確実性（高いほど内製化）

理論的予測（Williamson, 1985）:
  β₂₁ < 0 (資産特殊性が高いと外注しにくい)
  β₃₁ < 0 (資産特殊性が高いと合弁しにくい)
```

## III. 因果推論の方法論

### 3.1 因果推論の哲学的基盤

戦略研究における最も困難な課題は、相関関係から因果関係を識別することである。

**Pearlの因果階層（Pearl, 2009）**：

```
Level 1 (Association): 相関関係の観察
  → 「企業規模と業績は正に相関する」

Level 2 (Intervention): 介入効果の予測
  → 「企業規模を大きくしたら業績は向上するか？」

Level 3 (Counterfactual): 反実仮想の推論
  → 「企業規模を大きくしなかったら業績はどうだったか？」
```

戦略研究では、Level 2-3の因果推論が理論検証に不可欠。

### 3.2 内生性バイアスの問題

**内生性の3つの源泉**：

**1. 逆因果（Reverse Causality）**
```
問題: 業績 → 能力投資 の逆方向の因果も存在
解決: 操作変数法、パネルデータのラグ変数
```

**2. 省略変数バイアス（Omitted Variable Bias）**
```
問題: 能力と業績の両方に影響する第3の変数（経営者の質等）
解決: 固定効果モデル、操作変数法
```

**3. 測定誤差（Measurement Error）**
```
問題: 能力の真の値が観測できない
解決: 操作変数法、構造方程式モデリング
```

### 3.3 主要な因果推論手法

#### A. 操作変数法（Instrumental Variables, IV）

**原理**：
内生的説明変数Xの代わりに、Xと相関するが誤差項εとは無相関な変数Z（操作変数）を用いる。

**操作変数の条件**：
1. **関連性（Relevance）**: Cov(Z, X) ≠ 0
2. **外生性（Exogeneity）**: Cov(Z, ε) = 0

**戦略研究の例**：

```
問題: R&D投資 → 業績 の因果関係（内生性あり）

操作変数の候補:
  - 租税制度によるR&D税制優遇（外生的ショック）
  - 大学との地理的近接性（R&Dと相関、業績と直接無関係）

2段階最小二乗法（2SLS）:
  第1段階: R&D投資 = π₀ + π₁(操作変数) + ν
  第2段階: 業績 = β₀ + β₁(予測R&D投資) + ε
```

**注意点**：
- 弱い操作変数（F統計量<10）は使用不可
- 操作変数の外生性は理論的判断が必要

#### B. 差分の差分法（Difference-in-Differences, DiD）

**原理**：
処置群（政策を受ける）と統制群（受けない）の、処置前後の変化の差を比較。

**数学的表現**：
```
Yᵢₜ = β₀ + β₁(処置群) + β₂(処置後) + β₃(処置群×処置後) + εᵢₜ

β₃: DiD推定量（処置効果）
```

**戦略研究の例**：

```
研究問題: 企業統治改革が企業業績に与える影響

設定:
  処置群: 2015年に統治改革を実施した企業
  統制群: 統治改革を実施しなかった類似企業
  期間: 2013-2017年

結果:
  β₃ = 2.8** → 統治改革により業績が2.8%ポイント向上

前提条件（平行トレンド仮定）:
  処置がなければ、両群のトレンドは平行だったはず
  → イベントスタディで検証
```

#### C. 傾向スコアマッチング（Propensity Score Matching, PSM）

**原理**：
処置群と類似した特性を持つ統制群を選び出し、平均処置効果を推定。

**手順**：
```
1. 処置確率（傾向スコア）を推定
   P(処置=1|X) = f(X₁, X₂, ..., Xₖ)

2. 傾向スコアが近い個体をマッチング
   例: 最近隣マッチング、カーネルマッチング

3. マッチングされたサンプルで処置効果を推定
   ATT = E[Y₁ - Y₀ | 処置=1]
```

**戦略研究の例**：

```
研究問題: M&Aが買収企業の業績に与える影響

マッチング変数:
  - 企業規模、業種、業績、財務レバレッジ等

結果:
  ATT = 3.2%** → M&A実施企業は非実施の類似企業より
                   業績が3.2%ポイント高い

注意: 選択バイアスは観測変数でのみ統制
      観察されない要因（経営者の能力等）は残る
```

#### D. 回帰不連続デザイン（Regression Discontinuity Design, RDD）

**原理**：
ある閾値を境に処置の有無が変わる場合、閾値付近での不連続な跳びを処置効果とみなす。

**戦略研究の例**：

```
研究問題: 補助金受給が企業のイノベーションに与える影響

設定:
  - 補助金は評価点が80点以上の企業に支給
  - 評価点が79点の企業と81点の企業はほぼ同質と仮定
  
分析:
  イノベーション成果 = f(評価点) + τ(補助金受給) + ε
  
  閾値（80点）での不連続性 τ が処置効果
```

## IV. 統計ソフトウェアの比較と選択

### 4.1 主要ソフトウェアの特徴

| ソフトウェア | 強み | 弱み | 推奨用途 |
|------------|------|------|---------|
| **R** | 無料、柔軟性高、最新手法 | 学習曲線、速度 | 探索的分析、可視化 |
| **Stata** | 直感的、パネルデータに強い | 有料、拡張性低 | 確認的分析、因果推論 |
| **Python** | 汎用性、機械学習統合 | 統計専門性やや低 | 大規模データ、自動化 |
| **SPSS** | GUI、初心者向け | 柔軟性低、有料 | 基本的分析 |
| **Mplus** | SEM特化、複雑モデル | 有料、学習曲線 | 高度なSEM |

### 4.2 研究段階別の推奨

**探索的データ分析（EDA）**:
- **推奨**: R（ggplot2）、Python（seaborn）
- **理由**: 柔軟な可視化、迅速な試行錯誤

**記述統計と基本的回帰**:
- **推奨**: Stata、R
- **理由**: 簡潔なコマンド、標準的な出力形式

**パネルデータ分析**:
- **推奨**: Stata（xtreg, xtabond）
- **代替**: R（plm, lme4）
- **理由**: 豊富な診断ツール

**構造方程式モデリング（SEM）**:
- **推奨**: Mplus、R（lavaan）
- **理由**: 複雑なモデル仕様、適合度指標

**因果推論（IV, DiD, PSM）**:
- **推奨**: Stata（ivreg2, diff, psmatch2）
- **代替**: R（AER, MatchIt）
- **理由**: 標準的手法が実装済み

**大規模データ処理**:
- **推奨**: Python（pandas）、R（data.table）
- **理由**: メモリ効率、並列処理

### 4.3 実践的な選択指針

**初学者**:
- まず Stata で基本を習得
- 次に R で可視化と高度な手法を学習

**中級者**:
- Stata（確認的分析）+ R（探索的分析・可視化）の併用
- Python は必要に応じて

**上級者**:
- 研究目的に応じて最適なツールを選択
- 複数言語の統合（RからStataを呼び出し等）

## VI. よくある統計的誤りと対処法

### 6.1 p値の誤解

**誤り**: p<0.05 だから効果は大きい  
**正解**: p値は効果の大きさではなく、統計的有意性を示す

**対処**: 効果量（Cohen's d、R²）と実質的有意性を報告

### 6.2 多重検定問題

**誤り**: 20個の仮説を検定して1個が有意（p<0.05）→ 発見！  
**正解**: 偶然の有意性（Type I error）の可能性

**対処**: Bonferroni補正、FDR（False Discovery Rate）制御

### 6.3 過剰適合（Overfitting）

**誤り**: 説明変数を増やしてR²=0.95！  
**正解**: サンプル特有のノイズを学習しただけ

**対処**: 調整済みR²、クロスバリデーション、理論駆動の変数選択

### 6.4 サンプルサイズの無視

**誤り**: n=20で複雑なSEMを推定  
**正解**: サンプルサイズが不十分（経験則：変数×10以上）

**対処**: 事前の統計的検出力分析（Power Analysis）

### 6.5 測定尺度の無視

**誤り**: リッカート尺度を比率尺度として扱う  
**正解**: 順序尺度の性質を考慮

**対処**: 順序ロジット、非パラメトリック手法の検討

## VII. 統計的推論の限界と倫理

### 7.1 統計的有意性≠理論的重要性

統計的に有意でも、実質的に無意味な効果がある：
- 大サンプルでは微小な効果も有意になる
- 効果量、信頼区間、理論的意味を総合的に判断

### 7.2 p-hacking の禁止

結果が出るまで分析を繰り返すことは研究不正：
- 事前登録（Pre-registration）の推奨
- 探索的分析と確認的分析の明確な区別

### 7.3 出版バイアス

有意な結果のみが出版される傾向：
- null結果の報告も重要
- メタ分析による総合的評価

## まとめ

統計手法は、理論と証拠を結ぶ認識論的道具である。戦略・組織研究者は、単なる技術的手続きを超えて、各手法の理論的前提、適用条件、解釈上の注意点を深く理解する必要がある。

本ガイドで解説した手法は、研究目的、データ特性、理論的前提に応じて適切に選択され、組み合わされるべきである。そして何より重要なのは、統計的有意性だけでなく、理論的意味論、実質的重要性、方法論的厳密性の総合的判断である。

統計的推論の限界を認識しつつ、それを創造的に乗り越える知的営為こそが、実証研究の本質なのである。
