# ===================
# INSTALLATION GUIDE
# ===================
#
# Step-by-Step Installation:
#
# 1. Create virtual environment (RECOMMENDED):
#    python -m venv strategic_research_env
#    source strategic_research_env/bin/activate  # On Windows: strategic_research_env\Scripts\activate
#
# 2. Upgrade pip:
#    pip install --upgrade pip
#
# 3. Install core packages:
#    pip install pandas numpy scikit-learn statsmodels matplotlib seaborn
#
# 4. Install this requirements file:
#    pip install -r requirements_datamining.txt
#
# 5. Verify installation:
#    python -c "import pandas, sklearn, econml, xgboost, shap; print('âœ… All core packages installed')"
#
# Installation Size:
#   - Core packages: ~500 MB
#   - With ML (XGBoost, LightGBM, SHAP): ~800 MB
#   - With Causal Inference (EconML): ~1.2 GB
#   - With Deep Learning (PyTorch/TF): ~3-4 GB
#
# Installation Time:
#   - Core: 3-5 minutes
#   - With ML & Causal: 10-15 minutes
#   - Full suite: 20-30 minutes
#
# Platform Notes:
#   - macOS (M1/M2): All packages work natively with Apple Silicon
#   - Windows: Requires Visual C++ Build Tools for some packages
#   - Linux: Usually works without issues
#
# Troubleshooting:
#   - If EconML fails: pip install --upgrade cython numpy
#   - If XGBoost fails: conda install xgboost
#   - If LightGBM fails: conda install lightgbm
#   - If SHAP fails: pip install --upgrade setuptools wheel
#
# Tested on:
#   - Python 3.9, 3.10, 3.11
#   - macOS 13/14 (M1/M2, Intel)
#   - Windows 10/11
#   - Ubuntu 20.04/22.04
