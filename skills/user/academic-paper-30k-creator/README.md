# Academic Paper 30k Creator - クイックスタートガイド

## はじめに

このガイドは、Academic Paper 30k Creatorを初めて使用する方のための簡易マニュアルです。

---

## 5分でわかる使い方

### ステップ1: 重要度判定（2分）

研究テーマを4軸で評価：
- 影響範囲（25点）
- 革新度（25点）
- 実用性（25点）
- 複雑度（25点）

**合計80-100点** → 最高重要度（30,000-32,000字）  
**合計60-79点** → 高重要度（28,000-30,000字）  
**合計40-59点** → 中重要度（25,000-28,000字）

### ステップ2: 構成決定（3分）

標準8セクション構成を採用：
1. 序論（3,000字）
2. 先行研究（4,000字）
3. 理論的枠組み（3,500字）
4. 研究方法論（3,000字）
5. 実証分析（9,500字）
6. 理論構築（4,000字）
7. 考察（3,500字）
8. 結論（1,500字）

### ステップ3: 執筆開始（25分）

各セクションの必須要素チェックリストに従って執筆。  
同時にtokyo-citeで引用情報を記録。

### ステップ4: 統合・品質保証（8分）

- セクション統合
- 論理的整合性検証
- 引用同期（tokyo-cite）
- 最終チェック

## よくある質問

**Q: 時間内に完成しない場合は？**  
A: 重要度を1段階下げて文字数を削減してください。

**Q: 引用形式がわからない場合は？**  
A: tokyo-citeスキルのSKILL.mdを参照してください。

**Q: 論理的整合性に不安がある場合は？**  
A: logic-checker.mdの3段階検証を実施してください。
