## 🎯 研究の概要

### リサーチクエスチョン

1. **RQ1**: 市場成熟度の差異（超成熟市場、急成長市場、新興市場）は、プラットフォーム企業の事業戦略パターン（垂直統合戦略、エコシステム戦略、ハイブリッド戦略）とどのように対応しているのか？

2. **RQ2**: この市場成熟度と戦略パターンの適合性は、既存理論によってどのように説明されるのか？

### 分析対象企業（8社）

**第1層（超成熟市場）**:
- Alibaba/Tmall（中国）
- Coupang（韓国）

**第3層（急成長市場）**:
- Flipkart（インド）
- Amazon India（インド）

**第4層（新興市場）**:
- Sea Limited/Shopee（東南アジア）
- Tokopedia（インドネシア）

**第2層（成熟市場・比較事例）**:
- 楽天（日本）
- Amazon Japan（日本）

### 主要な発見

1. **超成熟市場**: ハイブリッド戦略が支配的
   - 物流・決済の統合 × エコシステム開放

2. **急成長市場**: エコシステム戦略が支配的
   - 第三者販売者の大規模動員

3. **新興市場**: ローカライズ型ハイブリッド戦略
   - 段階的垂直統合 × 地域別適応

### 理論的貢献

**統合的理論モデル**の構築：
- 制度的距離理論（Kostova, 1999）
- ダイナミック・ケイパビリティ理論（Teece et al., 1997）
- 組織的相補性理論（Milgrom & Roberts, 1995）

を統合し、市場成熟度と戦略適合性を説明する新しい理論的枠組みを提示。
