## 📋 提出前必須チェック項目

### ✅ Phase 1: 形式要件の確認（所要時間: 15分）

#### 1.1 論述形式
- [ ] 本文で箇条書き（•、-、1.、(1)等）を使用していない
- [ ] 参考文献リスト以外に箇条書きがない
- [ ] 複数項目の列挙は「第一に...、第二に...」等の論述形式を使用

#### 1.2 段落書式
- [ ] すべての本文段落が全角スペース1文字で字下げされている
- [ ] 見出しは字下げしていない
- [ ] 引用文は前後1行空け、全体を1文字字下げ

#### 1.3 文体
- [ ] である調（常体）で統一されている
- [ ] 「です・ます調」を使用していない
- [ ] 主観的表現（「思う」「感じる」）を避けている

#### 1.4 数値表記
- [ ] パーセント表記が統一されている（「51.1パーセント」または「51.1%」）
- [ ] 年号は西暦を使用（2024年）
- [ ] 金額は3桁区切り（「1兆5,000億円」）

#### 1.5 図表
- [ ] 図表に連続番号が付されている（図1、表1等）
- [ ] すべての図表にキャプションがある
- [ ] すべての図表に出典が明記されている
- [ ] 本文中で図表を参照している
