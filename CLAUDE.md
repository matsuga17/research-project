# CLAUDE.md - Research Project Configuration

## Project Overview

This is an academic research directory (研究) focused on strategic management, AI company analysis, and organizational studies.

## Directory Structure

- `/data/` - CSV datasets (financial data, market share, technical benchmarks, user statistics)
- `/databases/` - SQLite databases for corporate research
- `/skills/` - Claude Code skills and configurations
- `/投稿論文/` - Submission-ready papers
- `/再現用コード/` - Reproducibility code
- `/新松戸/` - Research materials
- `/GEO_Literature_Review/` - Literature review materials

## Research Topics

- AI企業成長戦略 (AI Company Growth Strategy)
- OpenAI戦略分析 (OpenAI Strategic Analysis)
- デジタルトレースデータと組織論 (Digital Trace Data and Organizational Theory)
- 日本家電産業 (Japanese Consumer Electronics Industry)
- 経営トップの脆弱性開示 (Executive Vulnerability Disclosure)

## Language Preferences

- Primary language: Japanese (日本語)
- Academic writing style
- Support bilingual content (Japanese/English)

## Working Guidelines

- Maintain academic rigor and proper citations
- Preserve existing file encodings (UTF-8)
- When editing markdown files, maintain Japanese formatting
- Data files are in CSV format with headers
